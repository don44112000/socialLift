# Get Started

Welcome to TikTok For Business Developers! With developers in mind, we make accessing and using our API For Business very easy. Only a few simple steps are required to gain access and start managing ads. **Below are the steps needed to take before making your first API call and some useful resources to help you quickly integrate with TikTok API for Business.**

# Step-by-step workflow 
> <span style="color: DodgerBlue">**Note**</span><br>The specific steps required for [authorization](https://business-api.tiktok.com/portal/docs?id=1781891256601602) and [authentication](https://business-api.tiktok.com/portal/docs?id=1781892384100354) will vary depending on the type of API you are using.

1. [Create a TikTok For Business account](https://ads.tiktok.com/marketing_api/docs?id=1738855099573250). 
2. [Register as a developer](https://ads.tiktok.com/marketing_api/docs?id=1738855176671234).
3. [Create a developer app](https://ads.tiktok.com/marketing_api/docs?id=1738855242728450).
4. [Authorization](https://business-api.tiktok.com/portal/docs?id=1781891256601602).
5. [Authentication](https://business-api.tiktok.com/portal/docs?id=1781892384100354).
6. Make your first API call: You can use [TikTok API for Business Postman collection](https://www.postman.com/tiktok/workspace/tiktok-api-for-business/folder/23207567-9c73b718-e802-40f0-a70d-db8524ed2494) to easily **test** your integration without impacting your real TikTok for Business account.
    > <span style="color: DodgerBlue">**Note**</span><br>You can use either API endpoints or SDK to make your first calls. See [SDK](https://ads.tiktok.com/marketing_api/docs?id=1764231376750658) to learn more details.
