# Step-by-step workflow

Below are the steps you need to take before making your first API call.
> <span style="color: DodgerBlue">**Note**</span><br>
The specific steps required for [authorization](https://business-api.tiktok.com/portal/docs?id=1781891256601602) and [authentication](https://business-api.tiktok.com/portal/docs?id=1781892384100354) will vary depending on the type of API you are using.
1. [Create a TikTok for Business account](https://business-api.tiktok.com/portal/docs?id=1738855099573250).
2. [Register as a developer](https://business-api.tiktok.com/portal/docs?id=1738855176671234).
3. [Create a developer app](https://business-api.tiktok.com/portal/docs?id=1738855242728450).
4. [Authorization](https://business-api.tiktok.com/portal/docs?id=1781891256601602).
5. [Authentication](https://business-api.tiktok.com/portal/docs?id=1781892384100354).
6. Make your first API call: You can use [TikTok API for Business Postman collection](https://www.postman.com/tiktok/workspace/tiktok-api-for-business/folder/23207567-9c73b718-e802-40f0-a70d-db8524ed2494) to easily **test** your integration without impacting your real TikTok for Business account.

> <span style="color: DodgerBlue">**Note**</span><br>You can use either API endpoints or SDK to make your first calls. **TikTok API for Business SDK is now in open beta. See [here](https://business-api.tiktok.com/portal/docs?id=1764231376750658) to learn more details.**
