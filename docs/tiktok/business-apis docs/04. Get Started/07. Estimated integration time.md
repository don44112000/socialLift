# Estimated integration time

This article introduces the estimated integration time needed for some common API integration use cases. If you have integration needs, the following information will help you learn about the average time needed and thus make integration plans. 
<br>Based on responses from hundreds of developers in a recent survey, we calculated the **average total integration hours** (the 2rd column in the table below) and **number of developers required** (the 3rd column in the table below) in the following use cases. For each integration, the total integration hours is a sum of working hours from all developers participating in that integration. For example, **Campaign Creation** takes an average total of 61.16 hours and requires 2.51 developers. 
> <span style="color: DodgerBlue">**Note**</span><br>The results are not absolute and only for reference. The integration time has to be considered on a case by case basis because the time needed is subject to many factors like developers' relevant product knowledge and previous integration experience.
