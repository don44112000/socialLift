# Use Cases

TikTok API for Business enables you to build functions according to your business needs, making advertising more automated, intelligent and customized. This chapter gives an overview of use cases for TikTok API for Business. 
- [Audience integration](https://ads.tiktok.com/marketing_api/docs?rid=t7d78487r&id=1758526010304514): You can use Audience API to create an audience and then apply it to your ad groups. 
- [Campaign creation](https://ads.tiktok.com/marketing_api/docs?rid=t7d78487r&id=1758526151905281): You can use Campaign Management API to create different types of ads, for example, Shopping Ads and Spark Ads. 
- [Lead generation](https://ads.tiktok.com/marketing_api/docs?id=1739289524101122): You can use Leads API or Subscription API to retrieve leads. 
- [Reporting](https://ads.tiktok.com/marketing_api/docs?id=1758528841973762): You can use Reporting API to run a synchronous or asynchronous report customized to your business needs. 
- [Webhook subscription](https://ads.tiktok.com/marketing_api/docs?id=1810521709205505): You can make subscriptions through webhooks to get real-time updates for leads, ad review status, ad fatigue status, or the Spark Ads authorization status of a video for a TCM order.
- [Media Mix Modeling data request](https://business-api.tiktok.com/portal/docs/media-mix-modeling-data-request/v1.3): You can use Media Mix Modeling (MMM) API to access MMM data for one or more ad accounts.
