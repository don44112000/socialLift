# Create Upgraded Smart+ Lead Generation Campaigns

This article walks you through the steps to create Upgraded Smart+ Lead Generation Campaigns.

Based on your preference for whether to use a catalog and optimization location, you can create an Upgraded Smart+ Lead Generation Campaign with optimization location as Instant Form, Upgraded Smart+ Lead Generation Campaign with optimization location as Website, Upgraded Smart+ Lead Generation Campaign with optimization location as TikTok direct messages, Upgraded Smart+ Lead Generation Campaign with optimization location as instant messaging apps, Upgraded Smart+ Lead Generation Campaign with optimization location as phone call, or Upgraded Smart+ Automotive Ads.

- To create an Upgraded Smart+ Lead Generation Campaign without using a catalog and with optimization location as Instant Form, see [Create an Upgraded Smart+ Lead Generation Campaign with Location as Instant Form](https://business-api.tiktok.com/portal/docs?id=1843324592169986).

- To create an Upgraded Smart+ Lead Generation Campaign without using a catalog and with optimization location as Website, see [Create an Upgraded Smart+ Lead Generation Campaign with Location as Website](https://business-api.tiktok.com/portal/docs?id=1843324606213121).

- To create an Upgraded Smart+ Lead Generation Campaign without using a catalog and with optimization location as TikTok direct messages, see [Create an Upgraded Smart+ Lead Generation Campaign with optimization location as TikTok direct messages](https://business-api.tiktok.com/portal/docs?id=1847302969710913).

- To create an Upgraded Smart+ Lead Generation Campaign without using a catalog and with optimization location as instant messaging apps, see [Create an Upgraded Smart+ Lead Generation Campaign with optimization location as instant messaging apps](https://business-api.tiktok.com/portal/docs?id=1847302988449921).

- To create an Upgraded Smart+ Lead Generation Campaign without using a catalog and with optimization location as phone call, see [Create an Upgraded Smart+ Lead Generation Campaign with optimization location as phone call](https://business-api.tiktok.com/portal/docs?id=1847302998941697).

- To create an Upgraded Smart+ Lead Generation Campaign using an Auto-Inventory catalog or Auto-Model catalog, see [Create Upgraded Smart+ Automotive Ads](https://business-api.tiktok.com/portal/docs?id=1843324618421314).
