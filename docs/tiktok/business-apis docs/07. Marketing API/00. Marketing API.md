# Marketing API

Marketing API enables developers to interact with [TikTok Ads Manager](https://ads.tiktok.com/i18n/login/) functionality at scale, allowing developers to programmatically query data, create and manage ads, and perform a wide variety of other tasks.

You can find more details of Marketing API in this chapter.

- [Overview](https://business-api.tiktok.com/portal/docs?id=1781891506567170): Introduce the concepts and benefits of Marketing API.
- [Get started](https://business-api.tiktok.com/portal/docs?id=1781891830241281): Introduces the steps needed before making your first Marketing API call.
- [Business Center](https://business-api.tiktok.com/portal/docs?id=1738864498580481): Introduces how to manage ad accounts, users, assets, and finance in Business Center.
- [Creatives](https://business-api.tiktok.com/portal/docs?id=1736271785207810): Introduces how to manage creative assets and view creative performance data.
- [Catalog Management](https://business-api.tiktok.com/portal/docs?id=1739578433846274): Introduces how to create and manage catalogs.
- [TikTok Store](https://business-api.tiktok.com/portal/docs?id=1739581901879298): Introduces how to create TikTok Store and manage storefront.
- [Campaign Management](https://business-api.tiktok.com/portal/docs?id=1735713781404673): Introduces how to create, manage, and optimize your campaigns, ad groups, and ads. 
- [Audience Management](https://business-api.tiktok.com/portal/docs?id=1739566496956417): Introduces how to create, update, and manage audiences.
- [Reporting](https://business-api.tiktok.com/portal/docs?id=1738864739862530): Introduces how to run a report and customize the data format per your needs.
- [Ad Measurement](https://business-api.tiktok.com/portal/docs?id=1739583617914881): Introduces how to measure the effectiveness of your ads.
