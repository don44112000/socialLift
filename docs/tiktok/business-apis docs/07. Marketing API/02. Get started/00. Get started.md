# Get started

Welcome to Marketing API! With developers in mind, we make accessing and using our API very easy. Only a few simple steps are required to gain access and start managing your TikTok marketing products. **Below are the steps needed to take before making your first API call and some useful resources to help you quickly integrate with Marketing API.**

# Step-by-step workflow 

1. [Create a TikTok For Business account](https://ads.tiktok.com/marketing_api/docs?id=1738855099573250). 
2. [Register as a developer](https://ads.tiktok.com/marketing_api/docs?id=1738855176671234).
3. [Create a developer app](https://ads.tiktok.com/marketing_api/docs?id=1738855242728450).
4. [Authorization](https://business-api.tiktok.com/portal/docs?id=1738373141733378).
5. [Authentication](https://business-api.tiktok.com/portal/docs?id=1738373164380162).
6. Make your first API call: You can use [TikTok API for Business Postman collection](https://www.postman.com/tiktok/workspace/tiktok-api-for-business/folder/23207567-9c73b718-e802-40f0-a70d-db8524ed2494) to easily **test** your integration without impacting your real TikTok for Business account.

# Useful resources
To discover other helpful resources that can assist you in getting started, such as rate limits, app permissions, and sandbox, please refer to this comprehensive [Get Started](https://business-api.tiktok.com/portal/docs?id=1735713609895937) Guide.
