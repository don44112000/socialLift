# Overview

Creative refers to multimedia elements presented to users in ads, such as videos and pictures. Creative requirements differ according to placements.  
Creative management mainly has the following parts:
- [Creative assets](https://ads.tiktok.com/marketing_api/docs?id=1739361739523073): Manage all kinds of materials used when creating advertisements, including videos, images, music, Instant Pages, interactive add-on, etc.
- [Creative tools](https://ads.tiktok.com/marketing_api/docs?id=1740286998081538): Provide tools such as Smart Fix, ads preview, etc. They can help you quickly create high-quality videos and optimize your videos on hand.
- [Creative reports](https://ads.tiktok.com/marketing_api/docs?id=1739067487319042): Help you view the performance data of the assets, and then make decisions on material optimization and advertisement placement according to these data.
