  - See [Cards](https://business-api.tiktok.com/portal/docs?id=1749019652141058) to check out how to use different cards in ads. 
- [Creative tools](https://ads.tiktok.com/marketing_api/docs?id=1740286998081538):  Discusses different creative tools and how to use these tools to improve ad performance. 
- [Creative reports](https://ads.tiktok.com/marketing_api/docs?id=1739067487319042): Discusses what kind of data insights we provide.
