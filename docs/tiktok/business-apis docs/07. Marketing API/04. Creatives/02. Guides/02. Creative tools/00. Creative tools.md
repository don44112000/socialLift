# Creative tools

Creative tools provide a variety of simple and common assets (including images, videos and ads) preview, generation and optimization capabilities.
- [Instant Page Editor SDK](https://business-api.tiktok.com/portal/docs?id=1740307202170881): Use it to create instant pages.
- [Ads Preview](https://ads.tiktok.com/marketing_api/docs?id=1740317791125506&rid=s4l9ixq576):  Runs in asynchronous mode. Use it to preview the actual display effect of the material in the placement after delivery.
- [CTA Recommendations](https://ads.tiktok.com/marketing_api/docs?id=1740307296329730&rid=s4l9ixq576): Get standard or dynamic CTA recommendations. These CTAs can be used in ads.
- [Smart Fix](https://ads.tiktok.com/marketing_api/docs?id=1741472523309058&rid=s4l9ixq576): Helps you identify and fix video issues, such as video size and resolution, which could later get your ads disapproved or rejected during campaign creation process.
- [Image Editing](https://ads.tiktok.com/marketing_api/docs?id=1739067107903489&rid=nmsgaz98b9): Runs in synchronous mode. Use it to edit an image to a size that you want.
- [Smart Text](https://ads.tiktok.com/marketing_api/docs?id=1739084248002626&rid=nmsgaz98b9): This tool provides advertisers with ad text suggestions based on their industries and products.
- [Creative Fatigue Detection](https://ads.tiktok.com/marketing_api/docs?id=1767568085608450): Helps you identify fatigued ads.
- [Share creative assets](https://ads.tiktok.com/marketing_api/docs?id=1773192725768193&rid=nmsgaz98b9): Share creative assets with other advertiser accounts.
