# Creative reports

Creative asset level reporting is essential to understand how an advertiser's creative assets resonate with their target audience. This information can then be used to improve future creatives and refine audiences. 


- [Creative basic reports](https://ads.tiktok.com/marketing_api/docs?id=1749023062758401): Help you get basic performance metrics at the creative asset level as well as creative asset attributes like height, width, and more for non-Spark Ads in one ad account.
- [Video Insights reports](https://business-api.tiktok.com/portal/docs?id=1789769972291585): Display the performance of multiple videos across one or more ad accounts, encompassing both Spark Ads and non-Spark Ads. 
- [Creative insights](https://ads.tiktok.com/marketing_api/docs?id=1749023065231362): Use data insights to create the best performing TikTok ad.
