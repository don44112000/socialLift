# Creative basic reports

Use [/creative/report/get/](https://ads.tiktok.com/marketing_api/docs?id=1740662135093314) to run a report of creative assets, and you will get basic performance metrics at the creative asset level as well as creative asset attributes like height, width, and more.  

To find out the supported metrics, see [Supported metrics in creative reports](https://ads.tiktok.com/marketing_api/docs?id=1739067551410178&rid=nmsgaz98b9).
