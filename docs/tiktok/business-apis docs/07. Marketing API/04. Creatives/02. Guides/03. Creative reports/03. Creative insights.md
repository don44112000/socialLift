# Creative insights

Creative insights help you create the best performing TikTok ad. We now provide three types of creative insights: benchmark data, second-level ad performance data, and Video Insights data.

# Benchmark data
You can use [/report/ad_benchmark/get/](https://ads.tiktok.com/marketing_api/docs?id=1738824501176321&rid=2n24t7nug9) to get performance data of ads against benchmarks. To find out the supported metrics, see [/report/ad_benchmark/get/](https://ads.tiktok.com/marketing_api/docs?id=1738824501176321&rid=2n24t7nug9). 

# Second-level ad performance data
You can use [/report/video_performance/get/](https://ads.tiktok.com/marketing_api/docs?id=1738825259075586&rid=2n24t7nug9) to get in-second performance data on ads. To find out the supported metrics, see [/report/video_performance/get/](https://ads.tiktok.com/marketing_api/docs?id=1738825259075586&rid=2n24t7nug9). 

# Video Insights data
You can also use [/report/video_performance/get/](https://ads.tiktok.com/marketing_api/docs?id=1738825259075586) to retrieve detailed Video Insights data on a specific video. 
 
Currently, we only support key frame metrics, such as CTR and conversions, which allow you to perform a second-by-second Key Frame Analysis to identify high-performance and low-performance frames in your video. With this information, you can optimize future videos by removing video elements that don't resonate with your audience and adding video elements that appeal to them.
 
To find out the supported metrics, see [/report/video_performance/get/](https://ads.tiktok.com/marketing_api/docs?id=1738825259075586). For more information about Video Insights, see [here](https://ads.tiktok.com/help/article/video-insights?redirected=2&lang=en).
