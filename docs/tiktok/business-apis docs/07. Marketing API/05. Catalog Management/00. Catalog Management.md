# Catalog Management

If you have a large volume of rapidly changing products, it is recommended to use the Catalog API to manage them. With this, advertisers can create, update, and delete multiple products in a single HTTP request.  

> <span style="color:DarkRed">**Important**</span><br>By using Catalog API endpoints, you agree to the [Product Catalog Terms](https://ads.tiktok.com/i18n/official/article?aid=6765817026859499525) and certify that you are abiding by the [TikTok Advertising Policies](https://ads.tiktok.com/help/article?aid=6685586866860720134). Please review these policies and ensure that the items you upload to your catalog don't breach them.
