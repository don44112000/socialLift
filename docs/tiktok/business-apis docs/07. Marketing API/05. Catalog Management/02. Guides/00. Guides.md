# Guides

This guide walks you through how to create and manage catalogs as well as some catalog related use cases. 
  - [Mapping between Catalog Management features in Catalog Manager and API configurations](https://business-api.tiktok.com/portal/docs?id=1782141580104705): Outlines the mapping between the Catalog Management features in Catalog Manager and the corresponding API configurations.
  - [Create catalogs and add products](https://ads.tiktok.com/marketing_api/docs?id=1754263112945665): Introduces how to create catalogs and add products, including creating a catalog, uploading products,  updating products, creating a product set, and binding event source to a catalog. 
  - [Troubleshoot catalogs with diagnostics](https://business-api.tiktok.com/portal/docs?id=1780803490136066): Introduces how to troubleshoot catalog products and event sources, including the Catalog Match Rate, by using Catalog Diagnostics.
    - [Mapping between Catalog Diagnostics features in Catalog Manager and API configurations](https://business-api.tiktok.com/portal/docs?id=1780803525320706): Outlines the mapping between the features for diagnosing catalog products and catalog event sources in Catalog Manager and the corresponding API configurations.
  - [Obtain Catalog Product Insights](https://business-api.tiktok.com/portal/docs?id=1805642743426050): Introduces how to obtain trending catalog products within your catalog and trending catalog product categories on TikTok.
    - [Mapping between Catalog Product Insights features in Catalog Manager and API configurations](https://business-api.tiktok.com/portal/docs?id=1805642902406145): Outlines the mapping between the Catalog Product Insights features in Catalog Manager and the corresponding API configurations.  
  - [Upload catalog videos to associate with catalog products](https://business-api.tiktok.com/portal/docs?id=1803654904260674): Introduces how to upload and manage catalog videos in your catalog.  
  - [Manage catalogs in a Business Center](https://ads.tiktok.com/marketing_api/docs?id=1754263100123137): Introduces how to manage catalogs as Business Center assets, including migrating catalogs to a Business Center and managing catalogs after migration. 
  - [Use catalogs when creating Shopping Ads](https://ads.tiktok.com/marketing_api/docs?id=1754263154550786): Introduces how to use catalogs to create Shopping Ads.
