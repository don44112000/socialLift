# TikTok Store

TikTok Store API enables you to retrieve first-party stores (TikTok Shops) under an ad account and products within TikTok Shops.
- To learn about TikTok Store and some common roles and concepts, see [Overview](https://ads.tiktok.com/marketing_api/docs?id=1763572749953025). 
- To find out the general workflow, see [Guides](https://ads.tiktok.com/marketing_api/docs?id=1739581925704769). 
- To find out what APIs are available for managing TikTok Store, see [API reference](https://ads.tiktok.com/marketing_api/docs?id=1740641357077505).
