# Guides

This guide walks you through the general workflow of managing TikTok Store via API. 

## Prerequisites

* The merchant has a TikTok business account. They can upgrade their TikTok account to TikTok business account. For instructions, refer to [this page](https://www.tiktok.com/business/en-US/apps/business-account).
* The merchant has a TikTok For Business account.
* The developer has registered as a developer on TikTok Marketing API portal and created a developer application. For details, see [Register as a developer](https://ads.tiktok.com/marketing_api/docs?id=1738855176671234) and [Create a developer app](https://ads.tiktok.com/marketing_api/docs?id=1738855242728450).

## Workflow

1. Create a Business Center. This needs to be completed on TikTok Ads Manager. For instructions, see [How to Create a TikTok Business Center](https://ads.tiktok.com/help/article?aid=12789).
2. Create a TikTok Shop in a Business Center or add an existing TikTok Shop to a Business Center. 
  - If you don't have a TikTok Shop, you can create a TikTok Shop in a Business Center. To learn about the steps, refer to [Create TikTok Shop from Business Center-Scenario B: Create my TikTok Shop](https://ads.tiktok.com/help/article/add-tiktok-shop-from-business-center?lang=en#anchor-9).
  - If you have created a TikTok Shop in TikTok Seller Center by following the steps in[ Set Up TikTok Shop Using TikTok Seller Center](https://ads.tiktok.com/help/article/set-up-tiktok-shop-using-tiktok-seller-center?lang=en), you can add an existing TikTok Shop to a Business Center. To learn about the steps, refer to [Create TikTok Shop from Business Center-Scenario A: Add my TikTok Shop](https://ads.tiktok.com/help/article/add-tiktok-shop-from-business-center?lang=en#anchor-4).
