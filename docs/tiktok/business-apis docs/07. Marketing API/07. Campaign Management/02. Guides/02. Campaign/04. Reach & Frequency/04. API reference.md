# API reference 

The Reach & Frequency API consists of the following endpoints:
- Manage R&F campaigns
  * [Get campaigns](https://ads.tiktok.com/marketing_api/docs?id=1739315828649986)
  *  [Create a campaign](https://ads.tiktok.com/marketing_api/docs?id=1739318962329602)
  *  [Update a campaign](https://ads.tiktok.com/marketing_api/docs?id=1739320422086657)
  *  [Update campaign status](https://ads.tiktok.com/marketing_api/docs?id=1739320994354178) 
- Manage R&F ad groups
  *  [Get ad groups](https://ads.tiktok.com/marketing_api/docs?id=1739314558673922)
  *  [Create R&F ad group](https://ads.tiktok.com/marketing_api/docs?id=1738235338194945)
  *  [Update R&F ad group](https://ads.tiktok.com/marketing_api/docs?id=1738235402874882)
  *  [Update ad group status](https://ads.tiktok.com/marketing_api/docs?id=1739591716326402) 
- Manage R&F ads
  *  [Get ads](https://ads.tiktok.com/marketing_api/docs?id=1735735588640770)
  *  [Create an ad](https://ads.tiktok.com/marketing_api/docs?id=1739953377508354)
  *  [Update an ad](https://ads.tiktok.com/marketing_api/docs?id=1739953405142018)
  *  [Update ad status](https://ads.tiktok.com/marketing_api/docs?id=1739953422970882) 
- Other R&F related endpoints
  *  [Get inventory estimates](https://ads.tiktok.com/marketing_api/docs?id=1738235487941633)
  *  [Cancel R&F ad order](https://ads.tiktok.com/marketing_api/docs?id=1740489099367425)
  *  [Get estimated information](https://ads.tiktok.com/marketing_api/docs?id=1740489551354881)
  *  [Validate contracts](https://ads.tiktok.com/marketing_api/docs?id=1740489855138818)
  *  [Get R&F time zone](https://ads.tiktok.com/marketing_api/docs?id=1740490147129345)
