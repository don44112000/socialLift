# Audience Management

Audience is an ad targeting option that enables TikTok For Business advertisers to target or exclude specific users when setting up ads. 
- To learn more about the introduction and benefits of the Audience API, see [Overview](https://ads.tiktok.com/marketing_api/docs?id=1746998266963969).
- To learn about different audience types and how to create and manage your audiences, see [Guides](https://ads.tiktok.com/marketing_api/docs?id=1746998271267841).
	- [Audience types](https://ads.tiktok.com/marketing_api/docs?id=1746998648212482): Introduces audience types and relevant APIs.
	- [About Streaming API](https://ads.tiktok.com/marketing_api/docs?id=1739566528222210): Introduces Streaming API as well as the differences between Customer File API and Streaming API and between the workflows of the two API services.
	    - To learn about how to normalize your identifiers for audience creation, see [Guidelines for identifier normalization](https://business-api.tiktok.com/portal/docs?id=1850030360880129).
	- [Create an audience](https://ads.tiktok.com/marketing_api/docs?id=1746998676042753): Introduces how to create audiences by using Audience API.
	- [Manage audiences](https://ads.tiktok.com/marketing_api/docs?id=1746998678415361): Introduces how to manage your audiences, including getting, updating, deleting, sharing audiences, applying audiences to multiple ad groups, etc.
	- [FAQs](https://ads.tiktok.com/marketing_api/docs?id=1746998681953281): Summarizes FAQs about the Audience Management API.
- To find out what APIs are available for managing audiences, see [API reference](https://ads.tiktok.com/marketing_api/docs?id=1739566530595841).
