# Overview

This section introduces Audience Management API and its benefits.

Audience is a group of users who may see your ads. Choosing the right audience can help you improve your ads' performance. You can create, update, and manage Custom and Lookalike Audiences through the Audience API to consistently have the most up-to-date audience being targeted.

> <span style="color:darkred">**Important**</span><br>By using the Audience API endpoints, you agree to the [TikTok Custom Audiences \(Customer File\) Terms](https://ads.tiktok.com/i18n/official/policy/custom-audience-terms).

> <span style="color: DodgerBlue">**Note**</span><br>To apply for access to the Streaming API (formerly known as Segment API), use [Allowlist Management](https://ads.tiktok.com/marketing_api/blog/detail/7102266224578396162) feature on the Developer Portal by selecting "Streaming API" from the drop down menu. We review applications on an ongoing basis. 

# Benefits of Audience API

* **Maximize your ROAS**: Schedule routine Custom Audience updates and creation to consistently have the most up-to-date audience being targeted.
* **Automate management process**: Automate the management of your Custom Audience builder through API endpoints. Automate CSV file upload without manual effort, and this can help minimize the risk of personal data breaches happening.
* **Maximize brand impact**: Dynamically update audiences to expand audience profile and build Lookalike Audience to find similar users on TikTok.
