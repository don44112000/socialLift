# Create an audience

This topic discusses how to create audiences by using [Audience](https://business-api.tiktok.com/portal/docs?id=1739940398505986) API.
- Custom Audience
<br>You can create a Custom Audience by [file](https://business-api.tiktok.com/portal/docs?id=1747012327159809) or [rules](https://business-api.tiktok.com/portal/docs?id=1739566532187138). A minimum audience size of 1,000 is required to target Custom Audiences in an ad group.
- Lookalike Audience
<br>You can use an existing audience to [create a Lookalike Audience](https://business-api.tiktok.com/portal/docs?id=1747012174934018) for retargeting purposes.
- Saved Audience
 <br>You can [create a Saved Audience](https://business-api.tiktok.com/portal/docs?id=1780156510696449) and then apply it to your ad groups.
