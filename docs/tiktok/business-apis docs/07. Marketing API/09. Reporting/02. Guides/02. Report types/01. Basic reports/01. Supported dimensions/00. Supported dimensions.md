# Supported dimensions

A dimension is an attribute to group your data by. Basic reports support the following dimensions. Please note that currently, we don't support all dimensions available on TikTok Ads Manager.

## Dimension grouping

The supported dimensions are categorized as ID, Time, Targeting, Asset, and Search dimensions. 
- ID dimensions: `advertiser_id`, `campaign_id`, `adgroup_id`,  `ad_id`, `ad_id_v2`.
- Time dimensions: `stat_time_day`, `stat_time_hour`.
- Targeting dimensions: `country_code`, `ad_type`.
- Asset dimension: `custom_event_type`,`event_source_id`, `page_id`, `component_name`, `room_id`, `post_id`, `image_id`, `video_material_id`, `minis_id`.
- Search dimension: `search_terms`, `search_keyword`, `match_type`.

See the following table to learn about the grouping rules for different dimension categories.
