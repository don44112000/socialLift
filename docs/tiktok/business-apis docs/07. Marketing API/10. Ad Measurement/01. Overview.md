# Overview

TikTok provides powerful event data connection tools to help you measure the effectiveness of your ads. Here is an overview of our event data connection tools. 


- App Events 
  - You can use [App Events API](https://ads.tiktok.com/marketing_api/docs?id=1701890978528258) or [TikTok App Events SDK](https://ads.tiktok.com/marketing_api/docs?id=1739583650172929) to share your app events. By doing so, you can use the App Events for targeting, measurement, and optimization purposes. 
  
- Website Events
  - For website advertisers, you can leverage both [Web Events API](https://ads.tiktok.com/marketing_api/docs?id=1739584809311234) and [Pixels](https://ads.tiktok.com/marketing_api/docs?id=1739583652957185) to measure the website visitor events.

- Offline Events
  - You can use [Offline Events API](https://ads.tiktok.com/marketing_api/docs?id=1758049779688450) to measure offline conversions from your ad campaigns.
  
# How to start
- Create a Mobile App with TikTok
	- Before creating your ads, you should use some endpoints to create and manage your mobile App. For more information, see [App Events](https://ads.tiktok.com/marketing_api/docs?id=1740857868662786).
- Implement App Events API or App Events SDK
	- To implement App Events API or App Events SDK, you should follow different steps to set up your ad account and developer account. For more information, see [Events API: App](https://ads.tiktok.com/marketing_api/docs?rid=fwo4q9l6gug&id=1739584805049345) and [TikTok App Events SDK](https://ads.tiktok.com/marketing_api/docs?id=1739583650172929).
- Implement Web Events API and Pixels
	- To implement Web Events API , you should first create a Pixel and generate an access token. For more information, see [Get Started with Pixel](https://ads.tiktok.com/help/article?aid=10021&lang=en&redirected=1), [Events API : Web - Get Started](https://ads.tiktok.com/marketing_api/docs?id=1739584855420929) and [Install Pixel using code](https://ads.tiktok.com/marketing_api/docs?id=1739585702922241).
- Implement Offline Events API 
  - To implement Offline Events API, you should first generate an access token and Create an Offline Event set. For more information, see [Events API: Offline - Get started](https://ads.tiktok.com/marketing_api/docs?id=1758051319816193).
