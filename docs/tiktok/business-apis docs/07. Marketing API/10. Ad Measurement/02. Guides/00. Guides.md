# Guides

This guide walks you through Events API, TikTok App Events SDK, Mobile apps and Pixel. 
	
*	**[Events API](https://ads.tiktok.com/marketing_api/docs?id=1739583647070209)**

	Use [App Events API](https://ads.tiktok.com/marketing_api/docs?id=1739584805049345), [Website Events API](https://ads.tiktok.com/marketing_api/docs?id=1739584809311234&rid=1ie3yzksjei), and [Offline Events API](https://ads.tiktok.com/marketing_api/docs?id=1758049779688450) to measure app events, website events, and offline conversions, and send them to TikTok for downstream processing.

* **[TikTok App Events SDK](https://ads.tiktok.com/marketing_api/docs?id=1739583650172929)**

	Integrate our Events SDK with your mobile apps so that events will be sent to TikTok.

* **[Mobile apps](https://ads.tiktok.com/marketing_api/docs?id=1739583651652673)**

	Manage the mobile apps that you want to promote. 

* **[Pixel](https://ads.tiktok.com/marketing_api/docs?id=1739583652957185)**

	Manage pixels that are used to measure the performance of your ads.
