# Events API Gateway

## **What is Events API Gateway**
The Events API Gateway is a simplified means of delivering server-side data to TikTok's Events API endpoint. Through the Pixel, advertiser event data is sent to TikTok's Pixel endpoint. Through the Events API, advertiser event data is sent from an advertiser's server to the TikTok Events API endpoint. This data can be more granular than the data otherwise sent via the Pixel. Via the Events API Gateway, a copy of the event data collected by the Pixel is sent to the TikTok Events API endpoint via an advertiser-administered cloud server.

The Events API Gateway empowers advertisers to adhere to best practices by seamlessly sending event data via both the TikTok Pixel and the Events API, reducing the need for specialized developer resources or third-party data partnerships. You will be able to send event data already configured in your TikTok Pixel, reducing the steps necessary to get started with server to server connections.

## **Why it matters now**

If you're relying only on browser-based tools, you're likely to miss key customer actions and leave campaign performance and ROI on the table.
