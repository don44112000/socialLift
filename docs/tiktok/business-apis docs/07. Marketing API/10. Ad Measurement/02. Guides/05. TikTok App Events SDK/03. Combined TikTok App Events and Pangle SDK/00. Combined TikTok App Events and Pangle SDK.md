# Combined TikTok App Events and Pangle SDK

## Overview
1. The TikTok App Events SDK version 7.7.0.0 (and above), available through this article, is a combined SDK that includes both the TikTok App Events SDK as a module and the Pangle Ad Monetization SDK as a module, that operate independently of each other. 

2. The entities providing these two SDK modules differ by region. Use of the TikTok App Events module is governed by the [TikTok For Business Commercial Terms of Service](https://ads.tiktok.com/i18n/official/policy/commercial-terms-of-service). Use of the Pangle Ad Monetization module is subject to separate legal terms  including as provided under the [Pangle Publisher Platform](https://www.pangleglobal.com/terms) or otherwise provided to you.

3. To learn more about the TikTok App Events module, please visit [About TikTok App Events SDK](https://ads.tiktok.com/help/article/about-the-tiktok-app-events-sdk). To learn more about the Pangle Ad Monetization module, please visit [About Pangle Ad Monetization SDK](https://www.pangleglobal.com/integration). 

4. You may choose to integrate this combined SDK by following the instructions provided in this article. The standalone TikTok App Events SDK continues to be offered through GitHub and the instructions provided in the rest of this documentation.

## About Pangle
Pangle is the ad network of TikTok for Business. Pangle enables global app developers to maximize earnings through exclusive TikTok for Business demand and advertisers to reach extensive audiences engaging in the premium mobile network. Learn more about how you can leverage Pangle to monetize your apps [here](https://www.pangleglobal.com/monetization).


## Supported programming languages
- For Android, the combined SDK currently supports Java & Kotlin.
- For iOS, the combined SDK supports Objective-C & Swift. 
- For Unity, the combined SDK supports C#.

## Getting started
- **If the Pangle SDK is already installed in your app,** follow the instructions in [Configuring app event reporting in existing Pangle SDK integrations](https://business-api.tiktok.com/portal/docs/configuring-app-event-reporting-in-existing-pangle-sdk-integrations/v1.3).
- **To install the combined SDK,** follow the instructions for your platform in [Android combined SDK installation](https://business-api.tiktok.com/portal/docs/android-combined-sdk-installation/v1.3) and [iOS combined SDK installation](https://business-api.tiktok.com/portal/docs/ios-combined-sdk-installation/v1.3).
