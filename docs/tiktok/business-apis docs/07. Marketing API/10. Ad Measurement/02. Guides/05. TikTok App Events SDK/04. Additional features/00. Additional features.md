# Additional features

- [**Advanced Predictive Measurement**](https://business-api.tiktok.com/portal/docs/advanced-predictive-measurement/v1.3)**:** Advanced Predictive Measurement enhances conversion measurement by leveraging additional device signals from the TikTok App Events SDK to improve attribution accuracy and ad delivery. The feature can be enabled or disabled directly in TikTok Events Manager without requiring an app update.
- [**Deeplinks and deferred deeplinks**](https://business-api.tiktok.com/portal/docs/deeplinks-and-deferred-deeplinks/v1.3)**:** Deferred deeplinks allow you to direct users to specific app pages after your app is installed and launched for the first time. You can add the following code after initialising the SDK to fetch and handle the deferred deeplink.
- [**Enhance data postback**](https://business-api.tiktok.com/portal/docs/enhance-data-postback/v1.3)**:** Enhance data postback automatically collects in-app page information such as metadata and button clicks through the App Events SDK. This feature is currently in limited closed testing with the objective of eventually improving ads delivery and providing signal automation improvements.
- [**In-app ad revenue reporting**](https://business-api.tiktok.com/portal/docs/in-app-ad-revenue-reporting/v1.3)**:** Starting from SDK version 1.5.0, in-app advertising mediation revenue reporting is supported.
- [**SKAdNetwork reporting and schema mapping**](https://business-api.tiktok.com/portal/docs/skadnetwork-reporting-and-schema-mapping/v1.3)**:** The App Events SDK supports Apple's SKAdNetwork (SKAN) framework for privacy-preserving attribution. Ensure that only one SDK in your app is responsible for updating SKAN conversion values.
