# Compliance best practices

- [**Limited data use**](https://business-api.tiktok.com/portal/docs/limited-data-use/v1.3): "Limited Data Use" (LDU) for the TikTok App Events SDK is a feature designed to give businesses more control over how their event data is used in TikTok's systems.
- [**Age gating**](https://business-api.tiktok.com/portal/docs/age-gating/v1.3): If your app **is** primarily marketed to children, an age gate is required. The TikTok App Events SDK must not be initialized and data must not be transferred to TikTok for Business until after the age gate is passed.
- [**Resources for data sharing after user consent**](https://business-api.tiktok.com/portal/docs/resources-for-data-sharing-after-user-consent/v1.3): TikTok provides tools that allow you to delay data transmission until the user has completed an in-app consent flow. This enables a delay prior to data transmission until consent is obtained.
