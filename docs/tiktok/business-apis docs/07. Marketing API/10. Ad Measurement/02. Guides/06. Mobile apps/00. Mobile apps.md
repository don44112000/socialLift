# Mobile apps

> <span style="color:DarkRed">**Important**</span><br>
> By using Events API endpoints, you agree to the [TikTok Business Products (Data) Terms](https://ads.tiktok.com/i18n/official/policy/business-products-terms) and that you will not share sensitive data with TikTok.

To promote mobile applications (App) with TikTok For Business, you need to create a mobile app in [TikTok Event Manager](https://ads.tiktok.com/help/article?aid=6754384850179850245) or via TikTok API for Business. Each app can be referenced by multiple ads under an advertiser account and it can be set as a promotion object. Apps mentioned in this article are not developer applications that are created on the TikTok API for Business portal, but mobile applications that you want to promote.

The following endpoints are available for you to manage apps.

* [Get app list](https://ads.tiktok.com/marketing_api/docs?id=1740859313270786)
* [Get app info](https://ads.tiktok.com/marketing_api/docs?id=1740859272887297)
* [Create app](https://ads.tiktok.com/marketing_api/docs?id=1740859280958466)
* [Update app](https://ads.tiktok.com/marketing_api/docs?id=1740859300069378)
* [Get app events](https://ads.tiktok.com/marketing_api/docs?id=1740859338750977)
* [Get app retargeting events](https://ads.tiktok.com/marketing_api/docs?id=1740859371033601)

`Note` 

* Application ID (App ID) in this article refers to the ID of the promoted app. It can only be obtained via the `/app/list/` endpoint.
* Apps cannot be deleted once created. They can only be modified.
* You can configure third-party data connection to monitor your ads' effectiveness, please refer to [the Help Cneter article](https://ads.tiktok.com/help/article?aid=6669666107177041925) for more details.
