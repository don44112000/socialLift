# Pixel

> <span style="color:DarkRed">**Important**</span><br>
> By using Events API endpoints, you agree to the [TikTok Business Products (Data) Terms](https://ads.tiktok.com/i18n/official/policy/business-products-terms) and that you will not share sensitive data with TikTok.

TikTok Pixel is a website event data connection tool provided by TikTok For Business. This tool makes it easier to measure ad performance by embedding a pixel code on a site. You can define multiple events to measure multiple user's behaviors under each pixel. The `Pixel ID` and `Pixel Event` have **a one-to-many relationship.**
