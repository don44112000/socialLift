# Organic API

Organic API enables developers to build tools for brands to manage their organic presence on TikTok and scale their TikTok efforts, leading brands to invest in their organic strategy and boosting it through paid ads (Spark Ads and non-Spark Ads).

Organic API consists of five APIs:
- [Accounts API](https://business-api.tiktok.com/portal/docs?id=1737565048641538): Seamlessly publish and manage native TikTok videos at scale, ensuring a consistent brand presence while freeing up valuable resources for strategic planning.
- [Mentions API](https://business-api.tiktok.com/portal/docs?id=1825103301081169): Understand how audiences are engaging with your brand by monitoring trending organic videos that discuss your products or services, enabling targeted engagement and reputation management.
- [TikTok One API](https://business-api.tiktok.com/portal/docs?id=1735827019995138): Simplify creator discovery and collaboration, reducing time spent finding the right partners while enhancing campaign authenticity through creator partnerships.
- [Discovery API](https://business-api.tiktok.com/portal/docs?id=1825127388184577): Quickly uncover trending hashtags and Commercial Music Library soundtracks, helping brands stay ahead of trends and create timely, relevant content with minimal manual research.
- [Spark Ads Recommendation API](https://business-api.tiktok.com/portal/docs?id=1825128359356417): Identify high-performing organic content that can be boosted into ads, driving traffic and conversions while leveraging the authenticity of organic content.
