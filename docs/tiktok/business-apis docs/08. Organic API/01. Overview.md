# Overview

TikTok's Organic API empowers brands with the critical insights they need to understand their organic performance on the platform and make informed marketing decisions. By providing valuable data on content trends, audience engagement, and brand mentions, brands can refine their organic strategies, identify opportunities for collaboration with creators, and determine which content deserves amplification through paid media. In essence, Organic API bridges the gap between organic success and paid growth, enabling brands to maximize their impact and ROI on TikTok.

# Use cases of Organic API 
You can improve your brand presence on TikTok by using Organic API in the following ways.

## Content ideation and optimization
1. **Accounts API - Comments**: Analyze user feedback about your brand from post comments.
2. **Mentions API**: Gather insights from user mentions in others' posts or comments.
3. **Discovery API**: Identify trending hashtags and Commercial Music Library (CML) sounds for content inspiration.
4. **Data Analysis**: Utilize insights to identify content overlaps for enhanced relevance.
5. **Accounts API - Post Publishing**: Streamline content publishing through automated job setups.
6. **Accounts API - Post Publishing**: Optimize post performance by incorporating trending hashtags and CML sounds.

## Community management
1. **Accounts API - Comments**: Analyze user feedback about your brand from post comments.
2. **Mentions API**: Gather insights from user mentions in others' posts or comments.
3. **Accounts API - Comment Management**: Ensure proactive and timely interaction with users to foster brand community by:
	- Responding to negative comments to manage brand reputation.
	- Engaging with positive comments to foster brand loyalty.

## Creator collaboration and content boosting
1. **Mentions API - Creator Discovery**: Identify influential users engaging with your brand.
2. **TikTok One API - Creator Discovery**: Discover top creators in your industry and explore their content.
3. **TikTok One API - Create Campaign**: Invite creators to participate in branded campaigns.
4. **Spark Ads Recommendation API**: Receive recommendations for boosting creator posts that are most likely to be high-performing.
5. **TikTok One API - Creator Post Performance**: Review and analyze creator post performance metrics.
