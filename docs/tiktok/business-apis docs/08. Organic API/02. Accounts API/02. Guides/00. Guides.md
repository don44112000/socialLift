# Guides

This guide walks you through the onboarding process and some useful resources of Accounts API. 


> <span style="color:DodgerBlue">**Note**</span><br>
> To ensure proper use of the Accounts API, starting March 20, 2026 at 00:00 (GMT+0), developers must complete the [Accounts API Access Application Form](https://bytedance.sg.larkoffice.com/share/base/form/shrlgu4WEvtSXpEDLcCw56u4Rfc) before submitting a new developer app or requesting a scope increase that includes the "TikTok Accounts" permission scope. Please follow the instructions carefully and provide clear, accurate information to help facilitate the application review process. For more details on Accounts API use cases, please refer to the [Accounts API](https://business-api.tiktok.com/portal/docs?id=1737944384433218) documentation.

- [Get started](https://business-api.tiktok.com/portal/docs?id=1760334598980610): Introduces the onboarding process and rate limits of Accounts API.
  - [Authorization](https://business-api.tiktok.com/portal/docs?id=1738083939371009): Introduces how to get authorization from businesses to manage their accounts.
   - [Authentication](https://business-api.tiktok.com/portal/docs?id=1738084387220481): Introduces how to get an `access_token` for subsequent API requests, as well as a `refresh_token` for access token renewal.
    - [Rate limits](https://business-api.tiktok.com/portal/docs?id=1738084416214017): Introduces the rate limits of Accounts API.

- [Manage URL properties](https://business-api.tiktok.com/portal/docs?id=1769324038780930): Provides information on how to manage URL properties for use with the Accounts API, including the types of URL properties that can be verified prior to publishing videos, the rules for ownership verification, and the step-by-step workflow for adding and verifying URL properties.
- [Manage comments on owned TikTok videos](https://business-api.tiktok.com/portal/docs?id=1776065099288577): Introduces the workflow and use cases of managing comments on owned TikTok videos.
- [Manage TikTok post ad authorization](https://business-api.tiktok.com/portal/docs?id=1815124785075201): Introduces the workflow of managing authorization codes of TikTok posts.
- [Accounts Insights data latency](https://business-api.tiktok.com/portal/docs?id=1746624508278786): Introduces data latency you might expect when using Accounts Insights API.
- [Webhooks](https://business-api.tiktok.com/portal/docs?id=1759977800177665): Introduces requirements and limitations for a webhook callback URL, available webhook events for Accounts API, and the webhook verification process. 
- [FAQs](https://business-api.tiktok.com/portal/docs?id=1776983576127490): Introduces common questions about Accounts API.
