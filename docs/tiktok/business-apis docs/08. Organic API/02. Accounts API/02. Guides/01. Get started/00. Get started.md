# Get started

Welcome to TikTok Accounts API! With developers in mind, we make accessing and using our API very easy. Only a few simple steps are required to gain access and start managing your TikTok Account. **Below are the steps needed to take before making your first API call and some useful resources to help you quickly integrate with TikTok Accounts API.** 


> <span style="color:DodgerBlue">**Note**</span><br>
> To ensure proper use of the Accounts API, starting March 20, 2026 at 00:00 (GMT+0), developers must complete the [Accounts API Access Application Form](https://bytedance.sg.larkoffice.com/share/base/form/shrlgu4WEvtSXpEDLcCw56u4Rfc) before submitting a new developer app or requesting a scope increase that includes the "TikTok Accounts" permission scope. Please follow the instructions carefully and provide clear, accurate information to help facilitate the application review process. For more details on Accounts API use cases, please refer to the [Accounts API](https://business-api.tiktok.com/portal/docs?id=1737944384433218) documentation.

## Step by step workflow
1. [Create a TikTok For Business account](https://ads.tiktok.com/marketing_api/docs?id=1738855099573250).
2. [Register as a developer](https://ads.tiktok.com/marketing_api/docs?id=1738855176671234).
3. [Create a developer app](https://ads.tiktok.com/marketing_api/docs?id=1738855242728450).
4. [Authorization](https://ads.tiktok.com/marketing_api/docs?id=1738083939371009).
5. [Authentication](https://ads.tiktok.com/marketing_api/docs?id=1738084387220481).
6. Make your first API call: You can use [TikTok API for Business Postman collection](https://www.postman.com/tiktok/workspace/tiktok-api-for-business/folder/23207567-9c73b718-e802-40f0-a70d-db8524ed2494) to easily **test** your integration without impacting your real TikTok Account.
