# Rate limits

The rate limit per authorized TikTok account and Accounts API endpoint is 40 queries per minute (QPM). Additionally, there is a separate rate limit for all Accounts API endpoints combined, which is determined by the global rate limit level assigned to your developer application. The following table outlines the rate limits for Accounts API endpoints combined per developer application based on the global rate limit level.

```xtable
| Global Rate Limit Level {30%} | QPM for All Accounts API Endpoints {60%}|
|---|---|
| Basic | 600 |
| Advanced | 1,000 |
| Premium | 1,000 |
| Ultimate | 1,000 |
```

To learn more about global rate limits and rate limit levels, see [Rate limits](https://business-api.tiktok.com/portal/docs?id=1740029171730433).
