# Webhook event types

TikTok offers two main categories of Webhook events related to TikTok accounts:

- Post publishing events. 
  - These events are related to the publishing of video or photo posts on a TikTok account. They are triggered when posts are published through specific API endpoints and provide updates on the status of published content throughout its lifecycle.
- Comment updates events.
  - These events are associated with comments on posts associated with your TikTok account. They notify you of new comments or changes to existing comments on your posts, allowing you to monitor and manage user interactions.
  
Each category serves a specific purpose, enabling you to receive real-time updates on different aspects of TikTok account activity and user engagement. By leveraging these webhook events, you can stay in sync with TikTok platform activities.

To learn more about how to create the configurations for these Webhook events , see [Subscribe to TikTok account Webhook events via Webhooks API](https://business-api.tiktok.com/portal/docs?id=1810521730276354).
