# Mentions API

Mentions API is a tool for monitoring mentions and analyzing community conversations towards your business on TikTok. By providing insights into community conversations, customer feedback, and social media trends, Mentions API can help you to improve your marketing strategies and build a stronger business presence on TikTok.
