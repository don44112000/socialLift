# Get started

Welcome to Mentions API! With developers in mind, we make accessing and using our API possible through only a few steps. 

# What are the criteria for discovering mentioned posts?
To appear in the search results, a post must meet the following requirements:
- User age: The post is not posted by a user who is underage or of an unknown age. For users in Europe, an age of 17 or under is considered underage, while for users in the US and other countries or regions, an age of 12 or under is considered underage.
- Post visibility: The post must be public.
- Lookback period: The post must be posted within the past 90 days.
- Mention: The post must explicitly mention (@) the business's handle in the caption.
- Rank: The post should be among the top 1000 most liked posts out of all the posts that mention the business's handle.

> <span style="color:DodgerBlue">**Note**</span><br>
> <ul><li>Self-mentioned posts, a business's own posts that meet the requirements above, will also appear in the search results.</li><li>After the account authorization is done, you need to wait for 24 hours before pulling mentions data.</li></ul>

3. [Create a developer app](https://business-api.tiktok.com/portal/docs?id=1738855242728450).
4. [Authorization](https://business-api.tiktok.com/portal/docs?id=1825103730490434).
5. [Authentication](https://business-api.tiktok.com/portal/docs?id=1825105003772930).
6. Make your first API call.
