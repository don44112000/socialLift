# Rate limits for Mentions API

Global rate limits are applicable to Mentions API endpoints. To learn about the global rate limits for a developer application, refer to [Rate limits - Global rate limits](https://business-api.tiktok.com/portal/docs?id=1740029171730433#item-link-Global%20rate%20limits).

We recommend limiting your calls to each Mentions endpoint for each TikTok Business Account to no more than one time per day. The data for the previous day will be refreshed daily between 0 am and 7 am UTC Time the following day. Therefore, we recommend pulling the data after 7 am UTC to ensure you have the most up-to-date information. Excessive calls to the endpoints within the same day are unnecessary.
