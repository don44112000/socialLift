# API reference

Mentions API is a tool for monitoring mentions and analyzing community conversations about your business on TikTok. By providing insights into community conversations, customer feedback, and social media trends, Mentions API can help you to improve your marketing strategies and build a stronger business presence on TikTok.

You can see [Guides](https://business-api.tiktok.com/portal/docs?id=1825103402321025) to learn more about the authorization and authentication procedure, rate limits, and more.
