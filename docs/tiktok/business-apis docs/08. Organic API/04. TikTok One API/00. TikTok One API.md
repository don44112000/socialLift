# TikTok One API

[TikTok One (TTO)](https://ads.tiktok.com/creative) is an all-in-one TikTok creative platform where you can access millions of creators, connect with production partners, and uncover creative insights — all in one place. 

TTO API allows you to access TTO Creator Marketplace creator data and discover creators to collaborate with on Branded Content videos.

Explore the documentation: 
- Getting started 
  - [Overview](https://business-api.tiktok.com/portal/docs?id=1763402336271362): Learn about the benefits and concepts of TTO API.
  - [Get started for TTO API](https://business-api.tiktok.com/portal/docs?id=1825117808886785): Understand the general workflow of TTO API.
- Implementation
  - [Rate limits for TTO API](https://business-api.tiktok.com/portal/docs?id=1825117901366274): Get familiar with the rate limits for TTO API.
  - [API reference](https://business-api.tiktok.com/portal/docs?id=1740302141934593): Explore the comprehensive documentation for all available TTO API endpoints. 
 
> <span style="color:DarkRed">**Important**</span><br>
> Starting June 30th, 2025, you can no longer use TikTok Creator Marketplace endpoints. These endpoints have been replaced by the new TikTok One API endpoints to provide you with better and more streamlined creator marketplace experiences. To ensure a smooth API integration, we recommend that you migrate to TikTok One endpoints as soon as possible.
