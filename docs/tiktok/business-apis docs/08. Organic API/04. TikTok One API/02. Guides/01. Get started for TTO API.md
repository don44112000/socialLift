# Get started for TTO API

With developers in mind, we make accessing and using our API possible through only a few steps. **Below are the steps needed to take before making your first API call and some useful resources to help you quickly integrate with TTO API.**

With just a few simple steps, you can start accessing and using the TikTok One (TTO) API. Below are the steps you need to complete before making your first API call, along with resources to help you quickly integrate with the TTO API.

## Step-by-step onboarding workflow
To get started with the TTO API, follow the following steps:
1. Set up TikTok for Business developer app.
	1. [Create a TikTok for Business account](https://business-api.tiktok.com/portal/docs?id=1738855099573250).
	2. [Register as a developer](https://business-api.tiktok.com/portal/docs?id=1738855176671234).
	3. [Create a developer app](https://business-api.tiktok.com/portal/docs?id=1738855242728450).<br>Once your app request is approved, reach out to your Partner Manager with the App ID to request access to TTO endpoints.
2. Configure TTO Creator Marketplace account.
	1. Sign up for a [TTO Creator Marketplace](https://ads.tiktok.com/creative/creatormarketplace) account.
	2. [Obtain authorization and authentication from your TTO Creator Marketplace account](https://business-api.tiktok.com/portal/docs?id=1825117875361793).
3. Collaborate with creators.
	1. Create a campaign and request for default Spark Ads authorization on the TikTok videos to be uploaded to the campaign by using [/tto/tcm/campaign/create/](https://business-api.tiktok.com/portal/docs?id=1815693549459458).
	2. Share the invite link with the creator.
	3. Creator uses the invite link to upload videos to TikTok.
4. Monitor performance.<br>Monitor video performance by using [/tto/tcm/report/](https://business-api.tiktok.com/portal/docs?id=1815693573340162).
5. [Create Spark Ads](https://business-api.tiktok.com/portal/docs?id=1739470744631298) using the uploaded TikTok videos.
