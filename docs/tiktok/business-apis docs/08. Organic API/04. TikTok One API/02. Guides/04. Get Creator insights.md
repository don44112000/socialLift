# Get Creator insights

The following endpoints are available for Creator insights. Note that the Public Insights and Search and Discovery Insights are only available for Creators within the TikTok One (TTO) Creator Marketplace. 

## Search and Discovery (TTO Creator Marketplace Creators Only)
- [Discover TTO Creators](https://business-api.tiktok.com/portal/docs?id=1825017307843585): Use this endpoint to search for Creators based on specific criteria. 

## Authorized Insights (All Creators)
- [Get Authorized Account Insights](https://business-api.tiktok.com/portal/docs?id=1735827374196738): Use this endpoint to get the authorized Account Level insights of the Creator.
- [Get Authorized Media Insights](https://business-api.tiktok.com/portal/docs?id=1735827480228865): Use this endpoint to get the authorized Video Level insights of the Creator.

## Public Insights (TTO Creator Marketplace Creators Only)
- [Get TTO Public Account Insights](https://business-api.tiktok.com/portal/docs?id=1828185358712897): Use this endpoint to get the basic profile information of a TTO Creator Marketplace Creator.
- [Get TTO Public Media Insights](https://business-api.tiktok.com/portal/docs?id=1828185411810305): Use this endpoint to get information about the videos of a TTO Creator Marketplace Creator.
