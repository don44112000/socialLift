# Business Messaging API

TikTok Business Messaging API is a suite of interfaces enabling authorized Business Accounts to exchange direct messages with TikTok users through API-integrated platforms. It allows businesses to seamlessly incorporate real-time messaging into their customer engagement strategies, supporting an organic-to-paid approach and enhancing brand presence on TikTok. 

> <span style="color:DodgerBlue">**Note**</span><br>
> TikTok Business Messaging API is **now in Open Beta in APAC, LATAM, METAP, and North America**. To learn more about regional availability, see [Access to Business Messaging API](https://business-api.tiktok.com/portal/docs?id=1832184145137922).
