# Get started

Welcome to Business Messaging API! With developers in mind, we make accessing and using our API possible through only a few steps.

## Step by step workflow
If you are new to API for Business, please complete the following steps before making your first API call. 
1. [Create a TikTok For Business account](https://business-api.tiktok.com/portal/docs?id=1738855099573250).
2. [Register as a developer](https://business-api.tiktok.com/portal/docs?id=1738855176671234).
3. [Create a developer app](https://business-api.tiktok.com/portal/docs?id=1738855242728450).
4. You [have obtained access to Business Messaging API](https://business-api.tiktok.com/portal/docs?id=1832184145137922).
5. [Authorization](https://business-api.tiktok.com/portal/docs?id=1832184159540418).
6. [Subscribe to Business Messaging Webhook events via Webhooks API](https://business-api.tiktok.com/portal/docs?id=1832190670631937).
7. [Authentication](https://business-api.tiktok.com/portal/docs?id=1832184175482945).
8. Make your first API call.

## API endpoints
To learn about available endpoints, see [API Reference for Business Messaging API](https://business-api.tiktok.com/portal/docs?id=1832184349193282).

## Rate limits
The rate limit for Business Messaging API endpoints is 10 queries per second (QPS).

## Return codes
To learn about the return codes for unsuccessful calls, see [Return codes for Business Messaging API](https://business-api.tiktok.com/portal/docs?id=1832184306529874).
