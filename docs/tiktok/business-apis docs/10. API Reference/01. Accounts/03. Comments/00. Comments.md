# Comments

This group consists of the following endpoints:
- [Get comments on an owned video](https://ads.tiktok.com/marketing_api/docs?id=1760232109619202)<br>
- [Get all replies to a comment](https://ads.tiktok.com/marketing_api/docs?id=1762228430145538)<br>
- [Create a new comment on an owned video](https://ads.tiktok.com/marketing_api/docs?id=1762228442072066)<br>
- [Upload a comment image](https://business-api.tiktok.com/portal/docs?id=1856212334897154)<br>
- [Reply to an existing comment on an owned video](https://ads.tiktok.com/marketing_api/docs?id=1762228448779266)<br>
- [Like/unlike an existing comment on an owned video](https://ads.tiktok.com/marketing_api/docs?id=1762228456608770)<br>
- [Hide/unhide an existing comment on an owned video](https://ads.tiktok.com/marketing_api/docs?id=1762228474532865)<br>
- [Delete an existing comment on an owned video](https://ads.tiktok.com/marketing_api/docs?id=1762228483080193)
