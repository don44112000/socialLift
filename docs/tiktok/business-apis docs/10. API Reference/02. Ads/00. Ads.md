# Ads

An ad is the smallest advertising unit and is the content presented to the target audience.

See [Guides-Ad](https://ads.tiktok.com/marketing_api/docs?id=1745292235571201) to find out the procedure for creating an ad and learn about the concept of identity. See [Ad structure](https://ads.tiktok.com/marketing_api/docs?id=1739381193120770) to find out the three advertising levels.

This group consists of the following endpoints.

* [Get ads](https://ads.tiktok.com/marketing_api/docs?id=1735735588640770)
* [Create ads](https://ads.tiktok.com/marketing_api/docs?id=1739953377508354)
* [Update ads](https://ads.tiktok.com/marketing_api/docs?id=1739953405142018)
* [Update the statuses of ads](https://ads.tiktok.com/marketing_api/docs?id=1739953422970882)
