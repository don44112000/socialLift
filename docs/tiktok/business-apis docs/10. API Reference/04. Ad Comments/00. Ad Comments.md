# Ad Comments

For standard video ads, or Spark Ads that are created by users who have linked to TikTok Business Accounts, if you have enabled commenting, you can manage the comments under the ads using Ad Comments API. You can get, update, reply to, delete, export comments, or update the statuses of comments.

You can also define a list of blocked words. If a comment contains a blocked word, it will be hidden to everyone except the user who made the comment. For the list of endpoints for blocked words, see [Ad Comments - Blocked Words](https://ads.tiktok.com/marketing_api/docs?id=1739029198660609).

This group consists of the following endpoints:
* [Get comments](https://ads.tiktok.com/marketing_api/docs?id=1738086301876225)
* [Get related comments](https://ads.tiktok.com/marketing_api/docs?id=1738086824859650)
* [Update the statuses of comments](https://ads.tiktok.com/marketing_api/docs?id=1738086844585985)
* [Reply to a comment](https://ads.tiktok.com/marketing_api/docs?id=1738957702411265)
* [Delete a comment](https://ads.tiktok.com/marketing_api/docs?id=1738957772267522)
* [Create a comment export task](https://ads.tiktok.com/marketing_api/docs?id=1738088144215041)
* [Get the status of a comment export task](https://ads.tiktok.com/marketing_api/docs?id=1738088242659330)
* [Download exported comments](https://ads.tiktok.com/marketing_api/docs?id=1738088341958658)
