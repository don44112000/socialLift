# Audience segment

Streaming API can be used to build target custom audiences for multiple advertisers at a time for both first and third party data. When you have high throughput needs or need the ability to make a batch request to the server by sending a stream of data when the audience information is ready, you can use this endpoint. **Data Partner companies are required to leverage the Streaming API.**  

><span style="color:DodgerBlue">**Note**</span><br>To apply for access to the Streaming API (formerly known as Segment API), use [Allowlist Management](https://ads.tiktok.com/marketing_api/blog/detail/7102266224578396162) feature on the Developer Portal by selecting "Streaming API" from the drop down menu. We review applications on an ongoing basis. 

To learn more about Streaming API and how it differs from Customer File API, see [Customer File API vs Streaming API](https://ads.tiktok.com/marketing_api/docs?id=1739566528222210). 

This group consists of the following endpoints.

* [Create/Delete an audience segment](https://ads.tiktok.com/marketing_api/docs?id=1739940583739393)
* [Map audience segments](https://ads.tiktok.com/marketing_api/docs?id=1739940585975809)
