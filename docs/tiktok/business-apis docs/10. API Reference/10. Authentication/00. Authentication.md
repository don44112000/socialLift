# Authentication

Authentication API enables you to generate or revoke long-term access tokens, and generate, renew, or revoke short-term access tokens.  

You can obtain long-term access tokens by receiving authorization from an ad account, a Business Center account, or a TikTok One Creator Marketplace account. In contrast, you need authorization from TokTok accounts to obtain short-term access tokens. 

To learn about the differences in the authorization and authentication processes across various APIs, see [Comparing authorization for different APIs](https://business-api.tiktok.com/portal/docs?id=1781891256601602#item-link-Comparing%20authorization%20for%20different%20APIs) and [Comparing authentication for different APIs](https://business-api.tiktok.com/portal/docs?id=1781892384100354#item-link-Comparing%20authentication%20for%20different%20APIs).

This group consists of the following endpoints:
* [Obtain a long-term access token](https://business-api.tiktok.com/portal/docs?id=1739965703387137)
* [Revoke a long-term access token](https://business-api.tiktok.com/portal/docs?id=1739965949088770)
* [Obtain a short-term access token](https://business-api.tiktok.com/portal/docs?id=1833997638479041)
* [Renew a short-term access token](https://business-api.tiktok.com/portal/docs?id=1833997679342594)
* [Revoke a short-term access token](https://business-api.tiktok.com/portal/docs?id=1833997665030145)
