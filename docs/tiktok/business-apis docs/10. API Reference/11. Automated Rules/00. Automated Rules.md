# Automated Rules

Automated Rules let you create rule-based tasks to automatically check and manage your campaigns, ad groups, and ads. To learn about how automated rules work on TikTok Ads Manager, see these articles:
* [About Automated Rules](https://ads.tiktok.com/help/article?aid=10005538)
* [How to Create and Apply Automated Rules](https://ads.tiktok.com/help/article?aid=10005539)
* [Automated Rule Conditions](https://ads.tiktok.com/help/article?aid=10005540)


This group consists of the following endpoints:

* [Create rules](https://ads.tiktok.com/marketing_api/docs?id=1738767670852610)
* [Get rules by ID](https://ads.tiktok.com/marketing_api/docs?id=1738768750822466)
* [Get rules by filters](https://ads.tiktok.com/marketing_api/docs?id=1738768861976578)
* [Get rule results](https://ads.tiktok.com/marketing_api/docs?id=1738769076676609)
* [Get result details](https://ads.tiktok.com/marketing_api/docs?id=1738769098314754)
* [Update rules](https://ads.tiktok.com/marketing_api/docs?id=1738769123170306)
* [Update the statuses of rules](https://ads.tiktok.com/marketing_api/docs?id=1738769144119298)
* [Bind/Unbind rules](https://ads.tiktok.com/marketing_api/docs?id=1738769164954626)
