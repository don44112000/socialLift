# Get rules by ID

Use this endpoint to get rules by rule IDs.

## Comparing v1.2 and v1.3
 ```xtable
 |Changes{30%}|v1.2{35%}|v1.3{35%}|
|---|---|---|
|Endpoint path| /v1.2/optimizer/rule/get/| /v1.3/optimizer/rule/get/|
|Request parameter data type |`advertiser_id`: number<br>`rule_ids`: number[]|`advertiser_id`: string<br>`rule_ids`: string[]|
|Response parameter data type|`dimension_ids`: number[]<br>`task_exec_id`: number<br>`rule_id`: number|`dimension_ids`: string[]<br>`task_exec_id`: string<br>`rule_id`: string|
|Response parameter deprecated in v1.3 |/|`recipients`|
```
## Request

**Endpoint** <https://business-api.tiktok.com/open_api/v1.3/optimizer/rule/get/>

**Method** GET

**Header**

```xtable
|Field|Data Type|Description|
|---|---|---|
|Access-Token {required}|string|Authorized access token. For details, see [Authentication](https://ads.tiktok.com/marketing_api/docs?id=1738373164380162).|
|Content-Type {required}|string|Request message type<br>Allowed format: `"application/json"`.  |
```

**Parameters**

```xtable
|Field|Data Type|Description|
|---|---|---|
|advertiser_id {required}|string| Advertiser ID |
|rule_ids {required}|string[]| IDs of the rules that you want to get|
```

### Example
```xcodeblock
(code JSON JSON)
{
    "advertiser_id": "75960823018",
    "rule_ids":["6982553361971232769","6984697088064290818"]
}
(/code)
```
