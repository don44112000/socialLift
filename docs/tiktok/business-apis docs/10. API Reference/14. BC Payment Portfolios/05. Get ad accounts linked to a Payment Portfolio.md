# Get ad accounts linked to a Payment Portfolio

Use this endpoint to retrieve the list of ad accounts linked to a Payment Portfolio.

## Request

**Endpoint** https://business-api.tiktok.com/open_api/v1.3/payment_portfolio/advertiser/get/

**Method** GET

**Header**

```xtable
|Field{30%}|Data Type{15%}|Description{55%}|
|---|---|---|
|Access-Token {Required}|string|Authorized access token. For details, see [Authentication](https://business-api.tiktok.com/portal/docs?id=1738373164380162).|
```

**Parameters**

``` xtable
|Field{30%}|Data Type{15%}|Description{55%}|
|--- |--- |--- |
|payment_portfolio_id{Required}|string|The ID of the Payment Portfolio.<br><br>To obtain a list of Payment Portfolio IDs, use [/payment_portfolio/get/](https://business-api.tiktok.com/portal/docs?id=1839151931301954).|
|page|integer|Current page number.<br><br>Default value: 1.<br>Value range: ≥ 1.|
|page_size|integer|Page size.<br><br>Default value: 10.<br>Value range: 1-50.|
```

### Example
```xcodeblock
(code curl http)
curl --location --request GET 'https://business-api.tiktok.com/open_api/v1.3/payment_portfolio/advertiser/get/?payment_portfolio_id={{payment_portfolio_id}}&page=1&page_size=50' \
--header 'Access-Token: {{Access-Token}}'
(/code)
```
