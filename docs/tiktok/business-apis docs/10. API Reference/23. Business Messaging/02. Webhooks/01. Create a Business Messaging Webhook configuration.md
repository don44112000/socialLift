# Create a Business Messaging Webhook configuration

Use this endpoint to create the configurations for Business Messaging Webhook events at the developer app level.

To learn more about how to create the configuration for these Webhook events , see [Subscribe to Business Messaging Webhook events via Webhooks API](https://business-api.tiktok.com/portal/docs?id=1832190670631937).

If you no longer wish to receive notifications, you can delete the Webhook configuration by using [/business/webhook/delete/](https://business-api.tiktok.com/portal/docs?id=1832184497985537).



## Request

**Endpoint** https://business-api.tiktok.com/open_api/v1.3/business/webhook/update/

**Method** POST

**Header**

```xtable
|Field{30%}|Data Type{15%}|Description{55%}|
|---|---|---|
|Content-Type {Required}|string|Request message type.<br>Allowed format: `application/json`.  |
```

**Parameters**

``` xtable
|Field{30%}|Data Type{15%}|Description{55%}|
|--- |--- |--- |
| app_id {Required} | string | ID of your developer application. <br><br>To find the <strong>App ID</strong>, navigate to <strong>[My Apps](https://ads.tiktok.com/marketing_api/apps/)</strong> > <strong>App Detail</strong> > <strong>Basic Information</strong>. |
| secret {Required} | string | Secret of your developer application. <br><br>To find the <strong>Secret</strong>, navigate to <strong>[My Apps](https://ads.tiktok.com/marketing_api/apps/)</strong> > <strong>App Detail</strong> > <strong>Basic Information</strong>. |
| event_type {Required} | string | The type of Webhook event that you want to subscribe to. <br><br> Enum value: <ul><li>`DIRECT_MESSAGE`: Business Messaging events.<ul><li>To learn more about the schema and sample payload of the events within this event category, see [Webhook schema](https://business-api.tiktok.com/portal/docs?id=1832190670631937#item-link-Webhook%20schema). </li></ul></li></ul>|
| callback_url {Required} | string | The callback URL. |
```

### Example

```xcodeblock
(code curl http)
curl --location --request POST 'https://business-api.tiktok.com/open_api/v1.3/business/webhook/update/' \
--header 'Content-Type: application/json' \
--data '{
    "app_id": "{{app_id}}",
    "secret": "{{secret}}",
    "event_type": "DIRECT_MESSAGE",
    "callback_url": "{{callback_url}}"
}'
(/code)
```
