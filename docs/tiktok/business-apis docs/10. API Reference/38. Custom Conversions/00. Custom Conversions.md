# Custom Conversions

Custom Conversions is TikTok’s innovative solution that enables advertisers to define precise conversion goals using existing Pixel, Events API, and App signals. This feature supports both Standard Events and advertiser-defined Custom Events, allowing for enhanced flexibility to ensure that campaign objectives are in line with actual business results.

This group consists of the following endpoints:
- [Get Custom Conversions associated with an event source](https://business-api.tiktok.com/portal/docs?id=1842225174460673)
- [Get the details of a Custom Conversion](https://business-api.tiktok.com/portal/docs?id=1842225206555074)
- [Create a Custom Conversion](https://business-api.tiktok.com/portal/docs?id=1842225226006530)
- [Update a Custom Conversion](https://business-api.tiktok.com/portal/docs?id=1842225247176705)
- [Delete a Custom Conversion](https://business-api.tiktok.com/portal/docs?id=1842225270315074)
