# Discovery

The Discovery API enables you to discover popular hashtags on TikTok so that you can create content around those topics, and uncover popular tracks for use in organic content and ad creation from the Commercial Music Library.

You can see [Overview](https://business-api.tiktok.com/portal/docs?id=1825127469285442) to learn more about the onboarding process of Discovery API.

This group consists of the following endpoints:
- [Get popular hashtags](https://business-api.tiktok.com/portal/docs?id=1825119032526849)
- [Get details of a popular hashtag](https://business-api.tiktok.com/portal/docs?id=1825119047070721)
- [Get trending videos related to hashtags](https://business-api.tiktok.com/portal/docs?id=1825119056527362)
- [Get popular tracks from the Commercial Music Library](https://business-api.tiktok.com/portal/docs?id=1825119063013505)
- [Get trending videos related to tracks](https://business-api.tiktok.com/portal/docs?id=1825119068941314)
- [Get trending search keywords](https://business-api.tiktok.com/portal/docs?id=1832798345014338)
- [Get recommended search keywords](https://business-api.tiktok.com/portal/docs?id=1832798361818114)
