# Create an app

Use this endpoint to create a mobile App. The created App can be used to [Create ads](https://ads.tiktok.com/marketing_api/docs?id=1739381193120770).
> <span style="color:DarkRed">**Important**</span><br>
> By using Events API endpoints, you agree to the [TikTok Business Products (Data) Terms](https://ads.tiktok.com/i18n/official/policy/business-products-terms) and that you will not share sensitive data with TikTok.
## Comparing v1.2 and v1.3 
 The following table outlines the differences between v1.2 and v1.3 endpoints. 
 ```xtable
|Changes{30%}|v1.2{35%}|v1.3{35%}|
|---|---|---|
|Endpoint path| /v1.2/app/create/| /v1.3/app/create/|
|Request parameter name|`partner_id` |`partner` |
|Request parameter data type|`advertiser_id`: number |`advertiser_id`: string |
|Response parameter data type|`app_id`: number |`app_id`: string |
```

## Request

**Endpoint** https://business-api.tiktok.com/open_api/v1.3/app/create/

**Method** POST

**Header**

``` xtable
|Field|Data Type|Description|
|---|---|---|
|Access-Token {Required}|string|Authorized access token. For details, see [Authentication](https://ads.tiktok.com/marketing_api/docs?id=1738373164380162). |
|Content-Type {Required}|string|Request message type.<br>Allowed format: `"application/json"`|
```

**Parameters**

``` xtable
|Field|Data Type|Description|
|---|---|---|
|advertiser_id {Required}|string| Advertiser ID |
|download_url {Required}|url|Application download link  |
|partner |string|Third-party tracking partners. <br> Enum values include: `AdBrix`, `Adjust`, `AppLink`, `AppsFlyer`, `DoubleClick`, `FlashTalking`, `Kochava`, `Other`, `ServingsSys`, `Singular`,`Tenjin`,`TlnkIo`. <br>For more information, see [Third Party Monitoring Partner](https://ads.tiktok.com/marketing_api/docs?id=1737174886619138).|
|tracking_url {+Conditional}|object|Required only in any of the following scenarios:<ul><li>The `partner` for the App is `Adjust`.</li><li>SAN (Self-Attributing Network) is not enabled on the `partner` for the App.</li></ul>Third party tracking partners [monitoring links](https://ads.tiktok.com/help/article?aid=6669666107177041925).|
#|click_url {+Conditional}|url|Click monitoring link, provided by your monitoring partner and can be copied to your website.|
#|impression_url |url|The default display monitoring link, the display monitoring link generated by your monitoring partner, can be copied to your website.|
```

### Example

``` xcodeblock
(code curl http)
curl --location  --request POST 'https://business-api.tiktok.com/open_api/v1.3/app/create/' \
--header 'Access-Token: {{Access-Token}}' \
--header 'Content-Type: application/json' \
--data '{
    "advertiser_id": "{{advertiser_id}}",
    "download_url": "{{download_url}}",
    "partner": "{{partner}}",
    "tracking_url": {
        "click_url": "{{click_url}}"
    }
}'
(/code)
```

``` xtable
|data |object|Return data. |
#|app_id |string|Mobile Application ID|

```
