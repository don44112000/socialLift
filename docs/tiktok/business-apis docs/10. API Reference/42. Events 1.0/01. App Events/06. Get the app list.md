# Get the app list

Use this endpoint to obtain the list of apps under your account.
> <span style="color:DarkRed">**Important**</span><br>
> By using Events API endpoints, you agree to the [TikTok Business Products (Data) Terms](https://ads.tiktok.com/i18n/official/policy/business-products-terms) and that you will not share sensitive data with TikTok.
## Comparing v1.2 and v1.3 
 The following table outlines the differences between v1.2 and v1.3 endpoints. 
 ```xtable
|Changes{30%}|v1.2{35%}|v1.3{35%}|
|---|---|---|
|Endpoint path| /v1.2/app/list/| /v1.3/app/list/|
|Request parameter data type|`advertiser_id`: number <br>`app_platform_ids`: number[]|`advertiser_id`: string <br>`app_platform_ids`: string[]|
|Response parameter name|`is_skan_allowed` |`skan_allowed` |
|Response parameter data type|`advertiser_id`: number <br>`app_id`: number<br>`tiktok_app_id`: number <br>`partner_id`: number  |`advertiser_id` : string <br>`app_id`: string <br>`tiktok_app_id`: string <br>`partner_id`: string |
|New response parameters|/|`self_attribution_enabled`<br>`advanced_dedicated_campaign_allowed`|
```

## Request

**Endpoint**  
https://business-api.tiktok.com/open_api/v1.3/app/list/

**Method** GET

**Header**

``` xtable
|Field|Data Type|Description|
|---|---|---|
|Access-Token {Required}|string|Authorized access token. For details, see [Authentication](https://ads.tiktok.com/marketing_api/docs?id=1738373164380162).|
```

**Parameters**

``` xtable
|Field|Data Type|Description|
|---|---|---|
|advertiser_id {Required}|string| Advertiser ID. |
|app_platform_ids |string[]| List of app platform ID for filter purpose.|
```

### Example

``` xcodeblock
(code curl http)
curl --get -H "Access-Token:xxx" \
--data-urlencode "advertiser_id=ADVERTISER_ID" \
https://business-api.tiktok.com/open_api/v1.3/app/list/
(/code)
```
