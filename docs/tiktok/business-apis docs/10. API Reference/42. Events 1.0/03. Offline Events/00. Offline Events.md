# Offline Events

> <span style="color:DarkRed">**Important**</span><br>
> By using Events API endpoints, you agree to the [TikTok Business Products (Data) Terms](https://ads.tiktok.com/i18n/official/policy/business-products-terms) and that you will not share sensitive data with TikTok.

Offline Events can be used to measure offline activity from people that see or interact with your ads. You can use this data to identify when conversions occur. Offline Events API enables you to create, update or delete an Offline Event set, get Offline Event sets under an ad account, and upload an Offline Event to an Offline Event set or upload Offline Events to an Offline Event set in bulk.

See [here](https://ads.tiktok.com/marketing_api/docs?id=1758051319816193) to learn more about how to create and manage Offline Event sets.

This group consists of the following endpoints:

* [Create an Offline Event set](https://ads.tiktok.com/marketing_api/docs?id=1758427576470529)
* [Update an Offline Event set](https://ads.tiktok.com/marketing_api/docs?id=1765596741157889)
* [Delete an Offline Event set](https://ads.tiktok.com/marketing_api/docs?id=1765596790860802)
* [Get Offline Event sets](https://ads.tiktok.com/marketing_api/docs?id=1765596808589313)
* [Report an Offline Event](https://ads.tiktok.com/marketing_api/docs?id=1758428013689857)
* [Report Offline Events in bulk](https://ads.tiktok.com/marketing_api/docs?id=1758428053652482)
