# CRM Events

> <span style="color:DarkRed">**Important**</span><br>
> By using Events API endpoints, you agree to the [TikTok Business Products (Data) Terms](https://ads.tiktok.com/i18n/official/policy/business-products-terms) and that you will not share sensitive data with TikTok.

CRM (Customer relationship management) integrations allow for leads generated on TikTok to directly transfer to an advertiser's CRM system. By postbacking lead information through CRM Events, you can help the lead optimization model identify more convertible leads for you to maximize ROI. CRM Events API enables you to get CRM Event sets within an ad account.

This group consists of the following endpoints:
- [Create a CRM Event Set](https://business-api.tiktok.com/portal/docs?id=1858830704718658)
- [Get CRM Event Sets](https://business-api.tiktok.com/portal/docs?id=1780896521680898)
