# Media Mix Modeling

Media Mix Modeling (MMM) is a measurement technique that leverages time-series modeling to understand the business impacts (prices) of all traditional and digital marketing touchpoints. By accounting for operations such as pricing, external factors, and economic factors, MMM can quantify ROI across an advertiser’s media channels and provide insight to future media spend planning.

To learn about how to access Media Mix Modeling data through API, see [Media Mix Modeling data request](https://business-api.tiktok.com/portal/docs/media-mix-modeling-data-request/v1.3).

This group consists of the following endpoints:
- [Create an MMM data request](https://business-api.tiktok.com/portal/docs/create-an-mmm-data-request/v1.3)
- [Check the status of an MMM data request](https://business-api.tiktok.com/portal/docs/check-the-status-of-an-mmm-data-request/v1.3)
- [Obtain the download URL for MMM data](https://business-api.tiktok.com/portal/docs/obtain-the-download-url-for-mmm-data/v1.3)
- [Get the MMM data request history](https://business-api.tiktok.com/portal/docs/get-the-mmm-data-request-history/v1.3)
