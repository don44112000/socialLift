# Music

Music is a type of creative assets, and you can see [Upload videos, images and music-Upload files](https://ads.tiktok.com/marketing_api/docs?id=1738963828459521) to learn about the procedure of finding suitable music. You can also use the music ID in creative tools such as [Smart Video](https://ads.tiktok.com/marketing_api/docs?id=1739352294717441), and [Smart Video Soundtrack](https://ads.tiktok.com/marketing_api/docs?id=1739089635852289).  In addition, music can be included in a video package via [/catalog/video_package/create/](https://ads.tiktok.com/marketing_api/docs?id=1740665273039873), where you would need to specify the music via `music_id`.

Music can be used in Carousel Ads. See [here](https://ads.tiktok.com/marketing_api/docs?id=1766217791987713) to learn more about how to create Carousel Ads.

This group consists of the following endpoints: 
* [Upload a piece of music](https://ads.tiktok.com/marketing_api/docs?id=1740052650395650)
* [Get the music list](https://ads.tiktok.com/marketing_api/docs?id=1740053909509122)
