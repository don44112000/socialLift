# Page

An instant page is a fast-loading lightweight web page that loads up to 11 times faster on TikTok than a standard web page. Page API enables you to retrieve the pages within an ad account, and create, validate or renew a TikTok Instant Page (TIP) Editor SDK access token. 

To learn more about different types of instant pages, see [Instant pages](https://business-api.tiktok.com/portal/docs?id=1739953260466178). To learn about how to use the TIP Editor SDK, see [Instant Page Editor SDK](https://business-api.tiktok.com/portal/docs?id=1740307202170881).

This group consists of the following endpoints:
- [Get the Page ID](https://business-api.tiktok.com/portal/docs?id=1820826387779586)
- [Create a TIP Editor SDK access token](https://business-api.tiktok.com/portal/docs?id=1820826755283969)
- [Validate a TIP Editor SDK access token](https://business-api.tiktok.com/portal/docs?id=1820826826000385)
- [Renew a TIP Editor SDK access token](https://business-api.tiktok.com/portal/docs?id=1820826847199234)
