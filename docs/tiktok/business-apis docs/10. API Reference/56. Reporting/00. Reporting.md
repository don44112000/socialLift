# Reporting

You can use Reporting APIs to run a report on your auction ads and reservation ads data, in synchronous mode or asynchronous mode. 

To find out an overview of Reporting API, see [Overview](https://ads.tiktok.com/marketing_api/docs?id=1751087777884161). To learn about synchronous and asynchronous reports, and different report types,  see [Guides](https://ads.tiktok.com/marketing_api/docs?id=1751087792726017). 
  - [Synchronous and asynchronous reports](https://ads.tiktok.com/marketing_api/docs?id=1751088469160001) introduces how to run synchronous and asynchronous reports.
  - [Report types](https://ads.tiktok.com/marketing_api/docs?id=1738864835805186) introduces the **use cases** of basic reports, audience reports, playable ads reports, DSA reports, Business Center reports, and GMV max ads reports, and their supported dimensions, metrics and filters. 
  - [Best practices](https://ads.tiktok.com/marketing_api/docs?id=1738864817670145) summarizes the best practices to  avoid or reduce time-outs in synchronous reports. 
  - [Data latency for reports](https://ads.tiktok.com/marketing_api/docs?id=1738864894606337) introduces the data latency you might expect when using Reporting API. 
  - [FAQs](https://ads.tiktok.com/marketing_api/docs?id=1738864850353154) summarizes common questions about Reporting API. 

This group consists of the following endpoints:
- [Run a synchronous report](https://ads.tiktok.com/marketing_api/docs?id=1740302848100353)
- [Create an asynchronous report task](https://ads.tiktok.com/marketing_api/docs?id=1740302766489602)
- [Get the status of an async report task](https://ads.tiktok.com/marketing_api/docs?id=1740302781443073)
- [Download the output of an async report task](https://ads.tiktok.com/marketing_api/docs?id=1740302808815618)
- [Cancel an asynchronous report task](https://ads.tiktok.com/marketing_api/docs?id=1803615367145537)
