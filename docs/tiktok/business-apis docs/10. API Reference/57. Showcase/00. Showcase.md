# Showcase

Showcase is a feature on TikTok that allows customers to shop directly from a seller’s or a creator’s account. You can find the Showcase tab when tapping the shopping bag icon on the profile page of a TikTok account. You can learn more about how to use Showcase [here](https://ads.tiktok.com/help/article?aid=10013245&redirected=1). Showcase API enables you to get the identities with Showcase permission under an ad account, and get the details of a Showcase, including the country or region the Showcase can target, and the available products in the Showcase.

See [Create Video Shopping Ads with products from Showcase](https://ads.tiktok.com/marketing_api/docs?id=1759232259360770) to learn about how Showcase is used to create Video Shopping Ads.

This group consists of the following endpoints:
- [Get identities with Showcase permission under an ad account](https://ads.tiktok.com/marketing_api/docs?id=1759233549899778)
- [Get the available regions for a Showcase via identity](https://ads.tiktok.com/marketing_api/docs?id=1759233561597954)
- [Get the available products in a Showcase](https://ads.tiktok.com/marketing_api/docs?id=1759233576199169)
