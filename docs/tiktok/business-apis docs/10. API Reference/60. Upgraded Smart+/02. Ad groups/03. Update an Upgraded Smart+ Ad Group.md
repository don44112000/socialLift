# Update an Upgraded Smart+ Ad Group

Use this endpoint to update an Upgraded Smart+ Ad Group.

This endpoint supports incremental updates.


## Request
**Endpoint** https://business-api.tiktok.com/open_api/v1.3/smart_plus/adgroup/update/


**Method** POST

**Header**

```xtable
|Field{35%}|Type{15%}|Description{50%}|
|---|---|---|
|Access-Token {Required}|string|Authorized access token. For details, see [Authentication](https://business-api.tiktok.com/portal/docs?id=1738373164380162).|
|Content-Type {Required}|string|Request message type.<br>Allowed value: `application/json`.|
```
