# Run an Upgraded Smart+ Creative Breakdown Report

Use this endpoint to retrieve an Upgraded Smart+ Creative Breakdown Report, a breakdown of the reporting data on creatives within Upgraded Smart+ Ads.

Upgraded Smart+ Creative Breakdown Reports support breakdown by time range (hour, day, week, or month).

## Request
**Endpoint** https://business-api.tiktok.com/open_api/v1.3/smart_plus/material_report/breakdown/

**Method** GET

**Header**
```xtable
|Field{30%}|Data Type{15%}|Description{55%}|
|---|---|---|
|Access-Token {Required}|string|Authorized access token. For details, see [Authentication](https://business-api.tiktok.com/portal/docs?id=1738373164380162).|
```

**Parameters**
```xtable
|Field{30%}|Data Type{15%}|Description{55%}|
|--- |--- |--- |
| advertiser_id {Required} | string | Advertiser ID. |
| dimensions {Required} | string[] | Dimension grouping conditions.<br><br>For example, `&#91;"main_material_id","stat_time_day"&#93;` indicates that both `main_material_id` and `stat_time_day` are grouped.<br><br>To learn about available dimensions and supported dimension groupings, see [Supported dimensions](https://business-api.tiktok.com/portal/docs?id=1843337879988226). |
| metrics | string[] | Metrics to query.<br><br>Default value: `&#91;"spend", "impressions"&#93;`.<br><br>Note that the amount of currency metrics is based on the currency of the ad account (`advertiser_id`).<br><br>To learn about the supported metrics at the ad level, see [Supported ad-level metrics for Upgraded Smart+ Creative Reports](https://business-api.tiktok.com/portal/docs?id=1843337909199938). |
| filtering | object | Filtering conditions. |
#| campaign_ids | string[] | A list of campaign IDs.<br><br>Max size: 200.<br><br>To obtain campaign IDs, use [/smart_plus/campaign/get/](https://business-api.tiktok.com/portal/docs?id=1843312818332930).<br><br><ul><li>Filter combination rule: You cannot combine the filter `campaign_ids` with the filter `adgroup_ids` or `smart_plus_ad_ids`.</li><li>Filter and dimension combination rule: You can combine the filter with any of the following dimensions:<ul><li>`advertiser_id`</li><li>`campaign_id`</li><li>`adgroup_id`</li><li>`smart_plus_ad_id`</li></ul></li></ul>  |
#| adgroup_ids | string[] | A list of campaign IDs.<br><br>Max size: 200.<br><br>To obtain ad group IDs, use [/smart_plus/adgroup/get/](https://business-api.tiktok.com/portal/docs?id=1843314879617026).<br><br><ul><li> Filter combination rule: You cannot combine the filter `campaign_ids` with the filter `adgroup_ids` or `smart_plus_ad_ids`.</li><li>Filter and dimension combination rule: You can combine the filter with any of the following dimensions:<ul><li>`advertiser_id`</li><li>`campaign_id`</li><li>`adgroup_id`</li><li>`smart_plus_ad_id`</li></ul></li></ul>|
#| smart_plus_ad_ids | string[] | A list of ad IDs.<br><br>Max size: 200.<br><br>To obtain ad IDs, use [/smart_plus/ad/get/](https://business-api.tiktok.com/portal/docs?id=1843317378982914).<br><br><ul><li>Filter combination rule: You cannot combine the filter `smart_plus_ad_ids` with the filter `campaign_ids` or `adgroup_ids`.</li><li> Filter and dimension combination rule: You can combine the filter with any of the following dimensions:<ul><li>`advertiser_id`</li><li>`smart_plus_ad_id`</li></ul></li></ul> |
#| main_material_types | string[] | A list of main creative types.<br><br>Enum values:<ul><li>`VIDEO_NON_SPARK_ADS`: Video (non-Spark Ads).</li><li>`VIDEO_SPARK_ADS`: Video (Spark Ads).</li><li>`CAROUSEL_NON_SPARK_ADS`: Carousel (non-Spark Ads).</li><li>`CAROUSEL_SPARK_ADS`: Carousel (Spark Ads).</li><li>`SINGLE_IMAGE_NON_SPARK_ADS`: Single image (non-Spark Ads).</li><li>`CATALOG_VIDEO`: Catalog video.</li><li>`CATALOG_VIDEO_TEMPLATE`: Catalog video template.</li><li>`CATALOG_CAROUSEL`: Catalog carousel.</li><li>`CATALOG_MULTI_SHOW`: Catalog Multi Show.</li></ul><br><ul><li>Filter combination rule: The filter `main_material_types` can be used alone.</li><li>Filter and dimension combination rule: You can combine the filter with any of the following dimensions:<ul><li>`advertiser_id`</li><li>`campaign_id`</li><li>`adgroup_id`</li><li>`smart_plus_ad_id`</li></ul></li></ul> |
#| main_material_id | string[] | A list of of main creative IDs.<br><br><ul><li>Filter combination rule: When you use this filter, you can specify up to 400 IDs in total for the filters `main_material_id`, `ad_text_entity_id`, `call_to_action_entity_id`, and `interactive_add_on_entity_id`.</li><li>Filter and dimension combination rule: You can combine the filter with any of the following dimensions:<ul><li>`advertiser_id`</li><li>`campaign_id`</li><li>`adgroup_id`</li><li>`smart_plus_ad_id`</li></ul> </li></ul> |
#| ad_text_entity_id | string[] | A list of ad text entity IDs.<br><br>The ID is a system-generated unique identifier for an ad text.<br><br><ul><li>Filter combination rule: When you use this filter, you can specify up to 400 IDs in total for the filters `main_material_id`, `ad_text_entity_id`, `call_to_action_entity_id`, and `interactive_add_on_entity_id`.</li><li>Filter and dimension combination rule: You can combine the filter with any of the following dimensions:<ul><li>`advertiser_id`</li><li>`campaign_id`</li><li>`adgroup_id`</li><li>`smart_plus_ad_id`</li></ul></li></ul> |
#| call_to_action_entity_id | string[] | A list of call-to-action entity IDs.<br><br>The ID is a system-generated unique identifier for a call-to-action.<br><br><ul><li>Filter combination rule: When you use this filter, you can specify up to 400 IDs in total for the filters `main_material_id`, `ad_text_entity_id`, `call_to_action_entity_id`, and `interactive_add_on_entity_id`.</li><li> Filter and dimension combination rule: You can combine the filter with any of the following dimensions:<ul><li>`advertiser_id`</li><li>`campaign_id`</li><li>`adgroup_id`</li><li>`smart_plus_ad_id`</li></ul></li></ul>  |
#| interactive_add_on_entity_id | string[] | A list of interactive add-on entity IDs.<br><br>The ID is a system-generated unique identifier for an interactive add-on.<br><br><ul><li>Filter combination rule: When you use this filter, you can specify up to 400 IDs in total for the filters `main_material_id`, `ad_text_entity_id`, `call_to_action_entity_id`, and `interactive_add_on_entity_id`.</li><li>Filter and dimension combination rule: You can combine the filter with any of the following dimensions:<ul><li>`advertiser_id`</li><li>`campaign_id`</li><li>`adgroup_id`</li><li>`smart_plus_ad_id`</li></ul> </li></ul> |
| start_date {Required} | string | Query start date (closed interval) in the format of `YYYY-MM-DD`.<br><br>The date is based on the ad account time zone. |
| end_date {Required} | string | Query end date (closed interval) in the format of `YYYY-MM-DD`.<br><br>The date is based on the ad account time zone.<br><br><p><span style="color:darkred"><b>Note</b></span>:<ul><li>The time difference between `start_date` and `end_date` must be equal to or less than 365 days.</li><li>If `dimensions` contains `stat_time_hour`, the time difference between `start_date` and `end_date` must be equal to or less than 7 days</li></ul></p> |
| sort_field | string | Sorting field.<br><br>Default sorting field: `spend`.<br><br>All supported metrics (excluding attribute metrics) support sorting. |
| sort_type | string | Sorting order.<br><br>Enum values: `ASC` (ascending), `DESC`(descending).<br>Default value: `DESC`. |
| page | number | Current page number.<br><br>Value range: ≥1.<br>Default value: 1. |
| page_size | number | Page size.<br><br>Value range: 1-100.<br>Default value: 10. |
```

### Example
```xcodeblock
(code curl http)
curl --location--request GET 'https://business-api.tiktok.com/open_api/v1.3/smart_plus/material_report/breakdown/?advertiser_id={{advertiser_id}}&dimensions=["main_material_id","ad_text_entity_id"]&metrics=["spend","clicks"]&filtering={"adgroup_ids": ["{{adgroup_id}}"]}&&start_date={{start_date}}&end_date={{end_date}}&page=1&page_size=100' \
--header 'Access-Token: {{Access-Token}}'
(/code)
```

```xtable
#| list|object[]|Data list.|
##| dimensions|object|Dimension data.|

###| main_material_type|string|Main creative type.<br><br>Enum values:<ul><li>`VIDEO_NON_SPARK_ADS`: Video (non-Spark Ads).</li><li>`VIDEO_SPARK_ADS`: Video (Spark Ads).</li><li>`CAROUSEL_NON_SPARK_ADS`: Carousel (non-Spark Ads).</li><li>`CAROUSEL_SPARK_ADS`: Carousel (Spark Ads).</li><li>`SINGLE_IMAGE_NON_SPARK_ADS`: Single image (non-Spark Ads).</li><li>`CATALOG_VIDEO`: Catalog video.</li><li>`CATALOG_VIDEO_TEMPLATE`: Catalog video template.</li><li>`CATALOG_CAROUSEL`: Catalog carousel.</li><li>`CATALOG_MULTI_SHOW`: Catalog Multi Show.</li></ul>|

###| call_to_action_entity_id|string|Call-to-action entity ID.<br><br>The ID is a system-generated unique identifier for a call-to-action.|
###| call_to_action_entity_type|string|Call-to-action entity type.<br><br>Enum values:<ul><li>`manual_call_to_action`: standard call to action.</li><li>`dynamic_call_to_action`: dynamic call to action.</li></ul>|
###| interactive_add_on_entity_id|string|Interactive add-on entity ID.<br><br>The ID is a system-generated unique identifier for an interactive add-on.|
###| interactive_add_on_entity_type|string|Interactive add-on entity type.<br><br>Enum value:<ul><li>`add_on`: interactive add-on.</li></ul>|
##| metrics|object|Metric data.|
###| &#123;metric_name&#125;|string|Returned metrics, which are determined by the `metrics` field specified in the request.|
###| multi_name|string|A combination of setting names based on the setting dimensions (`ad_text_entity_ids`, `call_to_action_entity_ids`, and `interactive_add_on_entity_ids`) specified in the request.<br><br>For instance, if you specify the dimensions `ad_text_entity_ids` and `call_to_action_entity_ids` in the request, the `multi_name` will be `&#123;ad_text_name&#125;_&#123;call_to_action_name&#125;`.<br><br>Example: `Video1_Adtext1_CTA1`.|

#| page_info|object|Pagination information.|
##| page|number|Current page number.|
##| page_size|number|Page size.|
##| total_number|number|Total number of results.|
##| total_page|number|Total pages of results.|
```
