# Supported dimensions for Upgraded Smart+ Creative Breakdown Reports

Dimension is an attribute to group your data by.

The following table outlines the available dimensions and supported dimension groupings in Upgraded Smart+ Creative Breakdown Reports.

```xtable
| Dimension {20%} | Dimension type {15%} | Description {20%} | Supported dimension groupings {45%} |
| --- | --- | --- | --- |
| `main_material_id` | Material dimension | Group by main creative ID. | <ul><li>`main_material_id` only</li><li>`main_material_id` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`)</li><li>`main_material_id` + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`)</li><li>`main_material_id` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`) + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`)</li></ul> |
| `ad_text_entity_id` | Setting dimension | Group by ad text entity ID.<br><br>The ID is a system-generated unique identifier for an ad text. | <ul><li>`ad_text_entity_id` only</li><li>`ad_text_entity_id` + `main_material_id`</li><li>`ad_text_entity_id` + one to two Setting dimensions (`call_to_action_entity_id`/ `interactive_add_on_entity_id`)</li><li>`ad_text_entity_id` + one to two Setting dimensions (`call_to_action_entity_id`/ `interactive_add_on_entity_id`) + `main_material_id`</li><li>`ad_text_entity_id` + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`)</li><li>`ad_text_entity_id` + one to two Setting dimensions (`call_to_action_entity_id`/ `interactive_add_on_entity_id`) + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`)</li><li>`ad_text_entity_id` + one to two Setting dimensions (`call_to_action_entity_id`/ `interactive_add_on_entity_id`) + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`) + `main_material_id`</li></ul> |
| `call_to_action_entity_id` | Setting dimension | Group by call-to-action entity ID.<br><br>The ID is a system-generated unique identifier for a call-to-action. | <ul><li>`call_to_action_entity_id` only</li><li>`call_to_action_entity_id`+ `main_material_id`</li><li>`call_to_action_entity_id` + one to two Setting dimensions (`ad_text_entity_id`/ `interactive_add_on_entity_id`)</li><li>`call_to_action_entity_id` + one to two Setting dimensions (`ad_text_entity_id`/ `interactive_add_on_entity_id`) + `main_material_id`</li><li>`call_to_action_entity_id` + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`)</li><li>`call_to_action_entity_id` + one to two Setting dimensions (`ad_text_entity_id`/ `interactive_add_on_entity_id`) + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`)</li><li>`call_to_action_entity_id` + one to two Setting dimensions (`ad_text_entity_id`/ `interactive_add_on_entity_id`) + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`) + `main_material_id`</li></ul> |
| `interactive_add_on_entity_id` | Setting dimension | Group by interactive add-on entity ID.<br><br>The ID is a system-generated unique identifier for an interactive add-on. | <ul><li>`interactive_add_on_entity_id` only</li><li>`interactive_add_on_entity_id`+ `main_material_id`</li><li>`interactive_add_on_entity_id` + one to two Setting dimensions (`ad_text_entity_id`/ `call_to_action_entity_id`)</li><li>`interactive_add_on_entity_id` + one to two Setting dimensions (`ad_text_entity_id`/ `interactive_add_on_entity_id`) + `main_material_id`</li><li>`interactive_add_on_entity_id` + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`)</li><li>`interactive_add_on_entity_id` + one to two Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`) + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`)</li><li>`interactive_add_on_entity_id` + one to two Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`) + one Time dimension (`stat_time_hour`/`stat_time_day`/`stat_time_week`/ `stat_time_month`) + `main_material_id`</li></ul> |
| `stat_time_hour` | Time dimension | Group by hour. | <ul><li>`stat_time_hour`+ `main_material_id`</li><li>`stat_time_hour` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`)</li><li>`stat_time_hour` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`) + `main_material_id`</li></ul> |
| `stat_time_day` | Time dimension | Group by day. | <ul><li>`stat_time_day`+ `main_material_id`</li><li>`stat_time_day` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`)</li><li>`stat_time_day` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`) + `main_material_id`</li></ul> |
| `stat_time_week` | Time dimension | Group by week. | <ul><li>`stat_time_week`+ `main_material_id`</li><li>`stat_time_week` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`)</li><li>`stat_time_week` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`) + `main_material_id`</li></ul> |
| `stat_time_month` | Time dimension | Group by month. | <ul><li>`stat_time_month`+ `main_material_id`</li><li>`stat_time_month` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`)</li><li>`stat_time_month` + one to three Setting dimensions (`ad_text_entity_id`/`call_to_action_entity_id`/ `interactive_add_on_entity_id`) + `main_material_id`</li></ul> |
```
