# Spark Ads Recommendation

Spark Ads Recommendation API empowers businesses and TikTok One (TTO) partners with tailored video recommendations for Spark Ads, strategically enhancing campaign impact for brands and creators.

You can see [Overview](https://business-api.tiktok.com/portal/docs?id=1825128393150530) to learn more about the general workflow and functionality of Spark Ads Recommendation API.

This group consists of the following endpoints:
- [Get Spark Ads video recommendations for a Business Account ](https://business-api.tiktok.com/portal/docs?id=1825128684907522)
- [Get Spark Ads video recommendations for a TTO account](https://business-api.tiktok.com/portal/docs?id=1825128700959746)
- [Create a campaign, an ad group, and a Spark Ad in one step](https://business-api.tiktok.com/portal/docs?id=1829744071179330)
