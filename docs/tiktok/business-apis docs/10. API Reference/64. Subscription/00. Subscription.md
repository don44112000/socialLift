# Subscription

Subscription API enables you to subscribe to ad account suspension status, leads, and more, get the details of a subscription, or cancel a subscription. 

See [Webhook subscription](https://ads.tiktok.com/marketing_api/docs?id=1810521739537409) to find out different types of webhook subscription that you can implement and read [Lead generation](https://ads.tiktok.com/marketing_api/docs?id=1739289524101122) to learn about how to create and test a lead subscription.

This group consists of the following endpoints:
* [Create a subscription](https://ads.tiktok.com/marketing_api/docs?id=1739092028876801)
* [Get subscription details of a developer app](https://ads.tiktok.com/marketing_api/docs?id=1739093832125442)
* [Cancel a subscription](https://ads.tiktok.com/marketing_api/docs?id=1739094758789122)
