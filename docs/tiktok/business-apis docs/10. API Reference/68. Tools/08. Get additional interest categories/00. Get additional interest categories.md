# Get additional interest categories

Use this group of endpoints to get additional interest categories (interest keywords). Note that only effective additional interest categories (`status` = `EFFECTIVE`) can be used in ad groups to target audiences.

- If you want to search for additional interest categories based on seed keywords, see [/tool/interest_keyword/recommend/](https://ads.tiktok.com/marketing_api/docs?rid=7xusz43oh3h&id=1763590884474882). 
- If you want to use IDs to get additional interest categories, see [/tool/interest_keyword/get/](https://ads.tiktok.com/marketing_api/docs?rid=7xusz43oh3h&id=1763590894544897).
