# Check Value-Based Optimization eligibility

Use this endpoint to check whether certain settings are eligible for Value-Based Optimization (VBO), and to get the available VBO bidding strategies for the specified settings.

To learn about how to enable VBO, refer to [Enable Value-Based Optimization](https://ads.tiktok.com/marketing_api/docs?id=1770019181843458).

><span style="color:DodgerBlue">**Note**</span><br> 
> If you specify through the request an allowlist-only VBO feature, and you are not allowlisted for the feature, the response will indicate that your settings are ineligible for VBO (`vo_status` = `NOT_SUPPORT`).


## Request

**Endpoint** https://business-api.tiktok.com/open_api/v1.3/tool/vbo_status/
