## Authorization acquisition

**Note**: All open API calls require [obtaining client authorization credentials](https://developers.tiktok.com/doc/client-access-token-management). When calling an open API, add the authorization credentials to the header. TikTok will verify the client's identity information based on the authorization credentials.
